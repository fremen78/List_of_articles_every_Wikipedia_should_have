== Tilmas n taganin ==


== Taganin timqqṛanin n umaḍal ==
Llant g umaḍal tugt n taganin imqquṛn, s izddar ra-nsiwel f 5 (Smmus) n taganin imqquṛn g umadal akk :


=== 1.Tagant n L-amazon ===
Tlla ɣ amenẓaw n Tamrikt n Wenẓul, Tajumma nns tga 5,000,000 Km²


=== 2.Tagant n Taïga ===
Tlla ɣ ugafa n umaḍal g amenẓaw n Sibirya.


=== 3.Tagant n Uglmam n L-congo ===
Tlla ɣ uglmam n L-congo g amenẓaw n Tafṛiqt, Tajumma nns tga 2,000,000 Km²


=== 4.Tagant n Daintree ===
Tlla ɣ amenẓaw n Uṣtralya.


=== 5.Tagant n umdlu ===
Tlla ɣ Tamurt n L-ikwattur, gis 1500 n wanawen n iɣersiwen
