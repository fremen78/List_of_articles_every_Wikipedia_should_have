Taswast neɣ Tafada d allal n usiden n wakud i wsiden n yizmaz d wussan d usuddes-nsen i yimesɣuṛa imettiyen, imesɣanen, ideblanen yakk d imsenziyen, anecta yettili-d s isebdaden yemgaraden deg teswasin yemgaraden, deg yakk taswasin-nni ttunefken kra n yismawen i kra n talliyin n uzemz anda tikwal ttilin d ussan neɣ d imalas neɣ d ayyuren neɣ d iseggasen, dɣa deg waṭas n tikkal ttilin d ayyuren yakk d iseggasen i teddun yakk d tuzzya n tfukt deg igenni yakk d wansa n tegnit deg tmezzit-is ɣef tfukt, neɣ teddun yakk d tlallit d taggara n wayyur akken daɣen illant teswasin yettganin ɣef tuzya n wayyur d tfukt d isin.
