Agaru irggign niɣ d Agaraw irggign iga agaru lli akkʷ yusɛan ɣ umaḍal, tawwṭfa n wakal nns tlkm gr 165,2 d 181,3 n umlyun n ukilumtiru amkkuẓ, ɣar waḥdu t igadda d unuḍu (double) n ugaru aṭlanṭiy, is yuti akkʷ mad tla taɣart, rnu is d agaru-ad bahra imqqurn yamu ma ittggan 46% n waman n wakal, ila yat tidri ilkmn ar 4280 n umitru.


== Ifi nns ==
Ɣ yat tsga ɣ Panama ar as ttinin "Iɣrm n wurɣ", issn usblyuniy Vasku Nunyiz di Balbuwa (1475-1517) ssɣ dar ayt tmazirt ann is nit illa yan ugaru ɣ tɣrdin n idrarn, argaz-ad d yan ukabaṛ gis 190 n uṣblyuni skrn ilmma yan uɣaras ɣ gr tẓgiwin ann llanin ɣ udrar aylliɣ nit ufan agaru-ad, ayad ɣ wassf n 26 Cutambir ɣ usggʷas n 1513. 
Tawafit n ugaru-ad ssɣ tsga n yilel ur rad tssnti amr d ussutl n Majillan (1480-1521) f umaḍal lliɣ t izgr ssɣ ixf ar ixf, nttan ad as ifkan tismt-ad n "Yilel irggign", acku dars ntta agaru-ad irggig f ugaru Aṭlanṭiy.
