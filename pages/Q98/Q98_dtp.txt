Lautan Pasifik nopo nga i bobos agayo om aralom id pialatan limo boogian rahat id winoun. Ii nopo nga mantad Lautan Artik id koibutan gisom Lautan Selatan (toi ko', tumanud do definisi, gisom hilo Antartika) id kabaatan, om rinontob do benua Asia om Australia id kotonobon om nogi Benua Amerika id kosilahon. 


== Pomungaranan ==
Ngaran rahat diti nopo nga naanu mantad do kopolombusan Mar Pacifico - "rahat di otoronong" i pinomungaranan di Ferdinand Magellan kopihondot do tongus otoronong ii nopurimanan om papasanang pelayaran dau mantad Selat Magellan kumaa Filipina.
