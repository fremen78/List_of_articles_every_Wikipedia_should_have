Jepun boros Jepun: 日本 Nippon/Nihon, ngaran poimbida' :日本国 Nipponkoku/Nihonkoku) nopo nga iso pomoguna kopulawan id Asia Kosilahon poingion id sampaping kotonobon Rahat Pasifik, id sampaping do kosilahon Rahat Jepun, om misombol do Republik Rakyat China, Korea Kabaatan om Rusia. Pulau di poinladsong id bobos koibutan Rahat Okhotsk, om watas id bobos kabaatan nopo nga piipiro pulau tokoro Rahat China Kosilahon id ponong kabaatan Okinawa. Kakadayan tagayo om ibu negara id Jepun nopo nga Tokyo, noomit miampai kakadayan tagayo suai miagal do Osaka, Yokohama, Nagoya, Sapporo, Fukuoka, Kobe om Kyoto.


== Pomungaranan ==
Ngaran "Jepun" nopo nga noolos mantad pomoroitan Sina miampai ngaran karajaan i sinuat id aksara Sina, 日本 miagal ko' nunu i sinuat do dudumagang Portugis id Melaka ontok abad kumo-16. Ngaran id aksari diti nopo nga suai mantad ko' pomoroitan boros di nopili: 

Japang, mantad pomoroitan boros Wu zeppen. Gunoon o pomoroitan diti id boros baku Indonesia miagal ko' Jepang.
Japun (ijaan Jawi laid: Templat:Rtl-lang) mantad kopomolombusan boros Min Boros Hokkien Ji̍t-pún, kopomolombusan diti nopo nga nopili montok boros baku suai kohompit do Boros Malaysia  sabaagi Jepun.

Ngaran「日本」i polombuson sabaagi nipon toi ko' nihon: Nipon nopo nga kopomolombusan rasmi i pinosuang id suang do tusin Yen, setem om pialaan sipoot sompomogunan om kopomoroitan do Nihon nopo nga tumanud do kopomoroitan timpu Edo i gunoon id suang do urusan au rasmi miagal do pibarasan sangangadau. Boros diti nopo nga kirati do "toud do tadau" toi ko' kosilahon mantad kinoyonon do China. Pomungaranan diti nokoimbulai asil do porutusan mobi Maharaja Jepun kumaa Dinasti Sui i momorintah id tana tagayo maamaso dii.
Sobolum nokopioput o Jepun miampai China, otutunan o pogun diti sabaagi do Yamato (大和). Maamaso timpu Tolu Kerajaan id China, otutunan o Jepun sabaagi pogun Wa (倭) toi ko' Wagoku (倭国).


== Sejarah ==


=== Prasejarah ===

Ponoriukan arkeologi pinopokito do Jepun nopo nga nion do tulun purba lobi kuang 50,000 toun di pogulu po, id timpu do Paleolitik Kolimpupuson. Kalapas nakawaya do piipiro timpu waig watu montok piipiro riong toun di nakatalin, koumbal di piipiro o Jepun do kopioput miampai tindal Asia maya do sunsuyon tindal (miampai Sakhalin id koibutan, om insan kanto Kyushu id kabaatan), gisom do haro koundolihon montok tulun, tongotayam, susumuni kumaa kepulauan Jepun mantad kinoyonon i baino nopo nga otutunan sabaagi Republik Rakyat China om nogi Korea. Timpu Paleolitik id Jepun nigi nga nakapaasil do kakamot winonsoi mantad watu tergilap di kumoiso id pomogunan, lobi kuang toun 30000 SM.
Miampai kinolimpupuson do timpu waig watu di kotohuri om kinorikatan do timpu di lobi talasu, koubasanan Jomon nokoimbulai lobi kuang 11000 SM, i kikawo do koposion humombo-monimung, i sampapas sedentari timpu Mesolitik gisom Neolitik om pamansayan do kakanan di bobos osopung id pomogunan. Notumbayaan do mogigion Jomon nopo nga monongodu kumaa tinaru Proto-Jepun om tinaru Ainu timpu baino.
Notimpuunan do timpu Yayoi lobi kuang 300 SM sabaagi panakatanda koimbulayan do teknologi-teknologi wagu miahal do mananom do parai id ranahon, i nogonop miampai sistem pomuwoigan om teknik pamansayan kakamot mantad bosi om gangsa i noit do rurumikot mantad China toi ko' Korea.
Id suang do sajara China, kumoinsan nopo tinaru Jepun narait id suang do ponuatan sajara laid, id suang do Buku Han i sinuat di toun 111. Timpu Yayoi nopo nga naalanan do timpu Kofun, lobi kuang toun 250 M, miampai negeri-negeri di nogonop miampai kagarasan tantara. Tumanud do Ponuatan Tolu Kerajaan, negeri di bobos nakalantoi id kopulauan Jepun maamaso di nopo nga Yamataikoku.


=== Zaman Klasik ===
Id suang do sajara Jepun nopo nga haro ponuatan di sinuat maamaso abad ko-5 om abad ko-6 Masihi, maamaso roromu koubasanan Cina miagal do Ponuatan Kanji om ugama Buddha noit do sinumuang id Jepun mantad kerajaan Baekja id Semenanjung Korea.

Koinlaaban do ugama Buddha id Jepun om inus ukir patung nopo nga ogumu narahung mantad ugama Buddha mantad China. Hali pia pihak pomorintah i mongigit do ugama Shinto, mangada kopio id kotimpuunon do kinorikatan ugama Buddha, nga maid-laid, yolo nopo nga minonorimo om pinopoinlaab do ugama Buddha id Jepun, gisom nosiliu sabaagi ugama di ogumu tutumanud id Jepun mantad po di timpu Asuka.
Id suang do Ponginwoguan Taika i pinoindalan di toun 645, noulud kaagu do Jepun o sistom pomorintahan dau miampai minomit do China, om iti nopo nga minongukan do ralan montok korotian Konfusianisme Cina pinopoinlaab do rahung diolo id Jepun gisom do abad ko-19.

Timpu Nara di abad ko-8 nopo nga tinanda' miampai pinolongkod kawagu o kuasa pomorintahan isoiso negeri id Jepun, miampai nowonsoi o ibu kota om istana maharaja id Heiji-kyo (baino nopo nga Nara). Maamaso timpu Nara, potilombus o Jepun do popoindalan pontadbiran miagal do China. Iso naanu diolo di bobos tagayo id gana sastara Jepun maamaso do timpu Nara nopo nga nogonop o buuk sajara Jepun i roiton sabaagi Kojiki (712) om Nihon Shoki (720).
Di toun 784, pinawaliu di Maharaja Kammu o ibu kota kumaa Nagaoka-kyō, om montadbir do hulo solinaid 10 no toun. Kalapas dii, pinoundaliu kawagu o ibu kota kumaa Heian-kyō (baino nopo nga Kyoto). Kinoundolihan ibu kota kumaa Heian-kyō nopo nga pinopotimpuun do timpu Heian, i otutunan sabaagi timpu kaamasan koubasanan klasik sandad id Jepun, lobi-lobi po id gana koinusan, hiis om kosusastaraan Jepun. Hikayat Genji i sinuat di Murasaki Shikibu om lirik sinding kabansaan Jepun Kimi ga Yo nopo nga mantad timpu Heian.


=== Timpu Pintangaan ===

Timpu pintangaan id Jepun nopo nga timpu feudalisme i tinanda' do piahawan kuasa do tulun pomorintah, i oilaan sabaagi kesatria om roitan sabaagi do samurai. Antakan di toun 1185, Minamoto no Yoritomo nopo nga narayou sabaagi shogun maya kinalantayon disido momorunsak do tinaru Taira i monolod do tinaru Minamoto (tinaru disio), om minaganu tonggungan sabaagi ponokotulu tantara i miilang do kuasa miampai Maharaja. Pomorintahan tantara i pinoingkakat di Minamoto no Yoritomo nopo nga roiton sabaagi Keshogunan Kamakura tu' longkod pomorintaan nopo nga id Kamakura (id kabaatan do Yokohama baino). Kalapas napatai i Yoritomo, minonguhup o tinaru Hōjō do keshogunan sabaagi shikken, miagal po do adipato montok tongosyogun. Kesyogunan Kamakura nopo nga nakaanu minongilag do serangan mantad Mongol, mantad wilayah China i maamaso dii nopo nga najajah do tinaru Mongol antakan di toun 1274 om 1281. Hali pia maya politik nopo nga oroitan sabaagi stabil, Keshogunan Kamakura nopo nga noontok do Ponondulian Kemmu (naala) do Maharaja Go-Daigo i pinopoguli kawagu kuasa id longon do maharaja. Maharaja Go-Daigo nopo nga naala di Ashikaga Takauji di toun 1336. Au nakaanu o Keshogunan Ashikaga minanahan do kagarasan ponguasa tantara om sanganu tana feudal (daimayo), mantad di, nokoimbulai o pisadangan poibpinayan di toun 1467 (Perang Ōnin) i pinopoimbulai do timpu iso abad i tinanda'an do pisangadan miampai paksi i rinoitan maamaso misangod o negeri-negeri toi ko' timpu Sengoku.
Antakan di abad ko-16, dudumagang om mubaligh Jesuit mantad Portugal nopo nga nokorikot di kumoinsan nogi id Jepun, om tinumimpuun do mogiolon-olon do padagangon om koubasanan id Jepun om nogi Pogun Kotonobon (Padagangan miampai Nanban). Tulun Jepun nopo antakan dii nga mamarait do tulun labus mantad Pogun Kotonobon sabaagi namban i kirati do tinaru gasar mantad kabaatan.

Oda Nobunaga minaganu do daimyo-daimyo suai montok kapansalan teknologi om senjata api mantad Eropah. Oruhai no nakalantoi i Nobunaga montok popiiso do Jepun sobolum pinatai id Kinaantakan Honnōji 1582. Sinolianan di Toyotomi Hideyoshi i Nobunaga, om pinopoindalan do tonggungan sabaagi popiiso Jepun di toun 1590. Kiangkab i Hideyoshi montok monguasai Korea, om induo pongumbalan montok manangod do Korea (Sangod Jepun kumaa Korea (1592-1598)), nga au nakalantoi tu naala id pisangadan miampai Korea, i nuhupan do Dinasti Ming. Kalapas napatai i Hideyoshi, minogidu o tantara disio mantad Semenanjung Korea di toun 1598.
Kalapas napatai i Hideyoshi, tanak kusai disio, i Toyotomi Hideyori minomoguno posisi disio sabaagi adipati montok Hideyori montok maganu sokodung politik om tentera mantad daimyo-daimyo suai. Kalapas minangala do mononokodung Hideyori id suang do Pisangadan Sekigahara di toun 1600, pinoingkakat i Teyasu sabaagi shogun di toun 1603. Pomorintahan tantara i pinoingkakat di Teyasu id Edo (baino lohowon sabaagi Tokyo) i lohowon sabaagi Keshogunan Tokugawa. Osuayan o Keshogunan Tokugawa kumaa kogiatan do mubaligh Katolik pinapasabap do noduhan o toinsanan piromutan miampai tulun-tulun Eropah. Pioputan padagangan sompomogunan nogi nga poinrontob kumaa dudumagang Belanda id Pulau Dejima, Nagasaki. Pomorintah Tokugawa nogi nga minomoguno mogikaakawo dasar suai miagal do undang-undang buke shohatto montok mongulud daimyo pointongkop watas. Di toun 1639, pinotimpuun do Keshogunan Tokugawa montok papagaraskuasa dasar sakoku ("pogun poinsompon") i nokoindalan solinaid duo satanga abad, om iti nopo nga rinoitan sabaagi do timpu Edo. Hali pia maamaso zaman poinsompon diti, kakal kasari kaanu o tulun Jepun minsingilo do toilaan mantad Pogun Kotonobon. Id Jepun, toilaan mantad buuk-buuk Kotonobon lohowon sabaagi rangaku (toilaan mantad Belanda), mantad piromutan do tulun Jepun om mogigion Belanda id Dejima, Nagasaki. Maamaso timpu Edo, tulun Jepun nogi nga nokotimpuun montok monoriuk pogun Jepun om minanahak istilah kokugaku montok ponoriukan pogun Jepun.


=== Timpu Nokoburu ===

Antakan 31 Gomot 1854,kinorikatan do Komodor Matthew Perry om "Kapal Toitom" Angkatan Rahat Amerika Syarikat minononsog do Jepun montok mongukab kumaa Pogun Kotonobon maya Pisokodungan Kanagawa. Pisokodungan-pisokodungan potilombus miampai pogun-pogun ponong Kotonobon id timpu Bakumatsu nopo nga pinopoimbulai do krisis ekonomi om politik id Jepun. Golongan samurai nopo nga monganggap do Keshogunan Tokugawa nopo nga opihoi no, om minomberontak gisom nokoimbulai o Pisangadan Boshin mantad di toun 1867 gisom 1868. Kalapas naala o Keshogunan Tokugawa, pinoguli o kokuasaaan kumaa maharaja (Ponondulian Meiji) om tinagal o sistom domain. Maamaso Ponondulian Meiji, minomoguno o Jepun do sistom politik, undang-undang om katantaraan mantad Pogun kotonobon. Pinohimagon o Kabinet Jepun do Dewan Penasihat Maharaja, mongulud Perlembagaan Meiji, om mombontuk Parlimen Diraja. Ponondulian Meiji nopo nga minongolon do Empayar Jepun sumiliu iso pogun industri nokoburu om pinapagaras do tantara dau gisom pinopoimbulai do konflik katantaraan maamaso mingoos do popoinlaab liputan wilayah dau id Asia. Kalapas minangala do China id suang do Sangod China-Jepun Kumoiso om Rusia id suang do Sangod Rusia-Jepun, nokuasa do Jepun o Taiwan, sampapas Sakhalin, om Korea.
Id kotimpuunon do abad ko-20, nakawaya o Jepun do timpu Taisho i pinokito sabaagi koingkakaton montok popoinlaab do wilayah om pamagarasan do tantara Jepun. Maamaso Pisangadan Pogun I, Jepun nopo nga id pihak Sekutu]] i nakalantoi do pisangadan dii, gisom nakaanu o Jepun do pinopoinlaab do rahung om wilayah kokuasaan dau. Potilombus o Jepun do popoindalan politik ekspansionis miampai minogion do Manchuria di toun 1931. Duo toun nakatalib, sinumoliwan o Jepun mantad Liga Bangsa-Bangsa kalapas nakaanu kecaman sompomogunan kokomoi pinogonuan do Manchuria. Antakan di toun 1936, minananda'longon o Jepun do Pakatan Anti-Comintern miampai Nazi Jerman, om miniamung miampai Jerman om Itali montok mombontuk do Paksi Sangod Pogun Kumoduo di toun 1941. 
Di toun 1937, sangod Jepun kumaa Manchuria pinopoimbulai do Pisangadan China-Jepun Kumoduo (1937-1945) i pinapasabap do noontok o Jepun do pangantaban miak mantad Amerika Syarikat Antakan 7 Momuhau 1941, minanangod o Jepun do longkod Tantara Rahat Amerika Syarikat id Pearl Harbor, om pinopokito do pisangadan miampai Amerika Syarikat, British, om Belanda. Pisangadan id Pearl Harbor minamagayat do AS id Pisangadan Pogun Kumoduo. Kalapas do kempen tantara id Rahat Pasifik, ogumu wilayah-wilayah Jepun di nangatagak i naanu id kotimpuunon do pisangadan. Binom do Amerika Syarikat o Tokyo, Osaka om kakadayan-kakadayan tagayo di suai. Kalapas pinaparatu do bom od Hiroshima om Nagasaki, minongkiala o Jepun miampai aiso syarat kumaa pihak berikat maamaso 15 Magus 1945 (Tadau Kinalantayon montok Jepun). 
Pisangadan nopo nga nakaanu minanansara do tulun Jepun om nogi wilayah-wilayah sangod do Jepun. Riong-riong do tulun nopo nga napatai id pogun-pogun Asia di naanu do Jepun, i popimpalak do slogan Kemakmuran Bersama Asia. Somok toinsanan industri om infrastruktur id Jepun nopo nga norunsak gama do pisangadan. Pihak Sekutu nopo nga pinopoindalan do repatriasi saragaagayo montok etnik Jepun mantad pogun-pogun Asia i koumbal do naanu Jepun. Tribunal Katantaraan Sompomogunan montok Kosilahon Tosodu i wowonsoyon do pighak Sekutu tumimpuun do 3 Mikat 1946, nolimpupusan miampai hukuman montok piipiro pemimpin Jepun i atalang kopio pinopoindalan do karaaton maamaso sangod.

Antakan di toun 1947, nokotorimo o Jepun do Perlembagaan wagu, hinonggo pinatatap do Jepun o pogundau sabaagi mongigit korotian pasifisme om popoindalan do demokrasi liberal. Poingion o As id Jepun nopo nga nolimpupusan miampai rasmi di toun 1952 miampai minananda'longon do Pibatasan San Francisco. Hali pia miagal di, tinimungan AS nopo nga kakal minanamong do longkod-longkod tapamgkal id Jepun, lobi-lobi no id Okinawa. Perserikatan Bangsa-Bangsa miampai rasmi nopo nga monorimo Jepun sabaagi anggota di toun 1956.
Kapalas Pisangadan Pogun Kumoduo, nakawaya o Jepun do kinoingkakatan ekonomi di osiau kopio, om pinopoili do Jepun sabaagi kagarasan ekonomi kumoduo tagayo id pomogunan, miampai koinlaaban koposoliwanan id suang do pogun kasar soginumu lobi kuang 10% sontoun solinaid apat dekad. Kosiahan do koingkakatan ekonomi Jepun diti nopo nga nolimpupusan di kotimpuunon toun 1990-an kalapas kinababakan do gelembung ekonomi dau.
