Ek programming language, ek rakam ke bhasa hae jon computer ke batae hae ki kon chij karo. Kuchh programming language hae; Python, Ruby, Java, Javascript, C, C++ aur C#.  Programming language se computer program likha jaae hae.
Jaada kar ke, programming language asli sabd ke kaam me laae hae ki insaan iske samjhe sake hae. Kuch programming language jaise ki Python aur JavaScript ke computer sidhe samaj le hae, lekin jaada me ii program ke compile kare ke parre hae jisse ii bhasa ke uu form me karaa jaae sake jon ki computer samjhe sake hae. Iske assembly language nai to machine language bola jaae hae.
