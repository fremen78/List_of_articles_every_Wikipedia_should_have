Computer software, jiske khaali software bhi bola jaae hae, jon ki computer ke batae hae ki ek taas ke kaise karaa jaae hae. Software me sab rakam ke program hae, jaise ki applications aur operating system. Application software ek khaas operation kare hae, jaise ki word processor. Operating system, applications aur computer ke biich me rahe hae. Kuch operating system hae Mac OS, Microsoft windows, Android, Linux.
