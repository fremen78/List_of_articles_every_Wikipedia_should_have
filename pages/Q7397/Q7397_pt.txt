Esta categoria é sobre todos os aspectos do software de computador (SW) que aqui é levado para incluir as seguintes categorias:

Categoria:Software aplicativo (Software aplicativo: office suites, word processors, spreadsheets, etc)
Categoria:Software de sistema (Software de sistema: operating systems, device drivers, desktop environments, etc)
Categoria:Ferramentas de programação de computador (Ferramenta de programação: assemblers, compilers, linkers, etc)
Os dois primeiros são classificados para a execução de software em computadores (ou seja, na verdade, usando os computadores), enquanto o último é sobre o desenvolvimento do software em primeiro lugar.
