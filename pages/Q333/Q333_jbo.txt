== tcesau kesyske ==
ni'o le clira mu'eske cu selgasnu le darno vecnu je bloti gidva noi ke'a lacri le nu le solri cu gidva be ke'a ti'u le nicte kei .e le tercange noi ke'a djuno le flecu be fa le rirxe fi le bancu korbi ku .e le crepu .e le nu ke'a nenri punji le tsiju le dertu sefta ri'a le murse solri tsani bo pixra .i ji'a le tcesau natmi cu zbasu la stonxeng. .e le mikce saske xislu pe le krasi merko prenu ri'a na djuno lakne ritli krinu
