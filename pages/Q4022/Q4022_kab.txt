Asif (Asget: isafen) d amsizel n waman agaman, Hraw yerna ɣezzif ttazalen dges waman id yettasen seg ugeffur d wedfel neɣ aman id yettali seg iɣbula d igelmimen, Dɣa aman n wasif tefakkan tikliwin-nsen deg ayen umi qqaren asafi send ma ad kecmen ar yilel neɣ agaraw


== Umuɣ n isafen ==
Asif n Nnil : D asif aɣezfan yakk deg umadal
Asif n Mississippi : Asif aɣezfan yakk di Tamrikt n Ugafa
Asif n Amaẓun : Asif ahrawan yakk deg umadal
Asif n Danub : Asif aɣezfan yakk deg umenẓawa n Turuft
Asif n Dejla d Furat : Sin n isafel yellan deg Asamar alemmas (Ɛiraq, Surya d Tturk)
Asif n Yangesti : Asif ɣezzifen deg Asya
Asif n Sabaw :  Asif deg tamurt n Iqbayliyen
Asif Asemmam (Aɛebbas) : Asif deg Tmurt n Yiqbayliyen
