Talwit d tagnit anda tamurt, ugdud neɣ awanak ur yella ara deg ṭṭrad, anamek-is daɣen d uneḥbus n ṭṭrad gar sin neɣ ugar n sin n yicenga.


== Ass agreɣlan n talwit ==
Deg 1981, Tuddsa n Yeɣlanen Yeddukklen tesnulfa-d ass agreɣlan n talwit. Ass agreɣlan n tawlit sfugulen deg 21 ctember yal assegas deg umaḍal akk.
