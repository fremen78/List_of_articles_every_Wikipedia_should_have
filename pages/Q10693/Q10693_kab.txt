Tafelwit tawalant d amsasa afelwan i yiferdisen ikruranen, ttwasnin s unect n wuṭṭun-nsen abelkim, tawila taliktṛunant d iylanen ikruranen imallusen, anda tamuska-ines tesskanay-d tizniwin tiwalanin. S umata, iferdisen i d-yezgan deg wedras amezwaru ttilin (tallit) d amɣuz ɣer tnila n uzelmaḍ, U d aramɣuz ɣer tnila n uyeffus, anda i ttmersen iferdisen ilan (yesɛan) tikliwin tikruranin timarwasin (yemcabin) deg yiwen n wejgu. Idrasen n tfelwit ttusemman (di tuget n tugnatin) s twalanin, ma d ijga ttusemman s tegrawin. Yerna 6 n tegrawin lant (sɛant) ismawen s tmerniwt ɣer wuṭṭunen usligen. Amedya, iferdisen n tegrawt tis 17 igan d ihalujinen, d tegrawt tis 18 igan d igazen iheggaṛen, akken i d-ittwafsar deg telɣa n kuẓ n yideggan usrimen neɣ deg yiswiren ifurkanen yeqqnen s tčarit n imezzayen ibelkamen yemgaraden.
Yella wamek ara ad tettwasuddes tfelwit tawalant i usuddem n wassaɣen gar n yiylanen n yiferdisen yemgaraden, yakk d d yiylanen d teklatin tikruranin yettuṛajun i yiferdisen ur d-nettwasnirem ara neɣ wid i d-ittwasuddsen s akken kan. Amusnaw n tekrura n Rusya Dmitri Mandelyiv yella d netta i d amdan amezwaru ifser-d tafelwit tawalant deg useggas n 1869, sin akin yella-d usnerni-ines s talɣa tagejdant i usegzi n tniliwin tiwalanin i yiferdisen yettwassnen imir-nni. Akken i icuyel kra n yiylanen iferdisen ur nettwafran ara i yellan yettuṛaju ad ččaren ibeṛjujen n tfelwit. Dɣa yella-d usihrew n tekti n Mandelyiv u tettusenṭeḍ yakked usnirem d usnulfu n yiferdisen imaynuten niḍen d usnerni n tmudmiwin tiẓranin timatuyin i usegzi n tkelli takrurant.
Tafelwit tawalant tatrart tettak-d akatar yelhan i teslaḍt n tsedmirin tikruranin, yerna mazal-itt tettusexdem s waṭas di tekrura d tfizikt taɣisant d tussniwin niḍen.

Tigrawin:
Tagrawt d ajgu abedday (aratak) di tfelwit tawalant n iferdisen. Di tfelwit-a llant 18 n tegrawin. Iferdisen yellan di yal tagrawt lan yiwen n usuddes n tbessast taliktṛunant eg wayen yezzin ɣef umḍan n iliktṛunen, dɣa d anecta i yettaken i yiferdisen-a amcabi n yiylanen.
