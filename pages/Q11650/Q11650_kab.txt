Iliktrunik neɣ Taliktrunit, d aḥric n tasengama yewwi-d tikli d isemda n ilikṭrunen akked allalen iliktrunikanen.
Tadiyudt, Atranzisṭur ed Asedfay d iferdisen n llsas n ṛṛadyu, tiliẓri ed aselkim.


== Amezruy ==
Almad n trisiti deg leqrun wis 18 d 19, i yeǧǧan usnerni n tnaεurin n trisiti. Amezruy n iliktrunik tebda ad tamhezt deg tagara n lqern wis 19 anda amesfizik agnizi Sir Joseph John Thomson isersen ulikṭrun ed  amesfizik amarikani Robert Millikan isersen aktili (leḥsab) n tirint n uliktrun.


== Iliktrunik deg tutlayin timaziɣin ==
Deg 2011, Asqamu Unnig n Timmuzɣa yessiẓreg amawal n Taliktrutiknikt, Tafrensist-Tamaziɣt (DICTIONNAIRE d‟électrotechnique Français-Tamazight) tebna ɣef taqbaylit em tameslayt deg llsas, amawal-nni yesɛa ugar 3600 yisumuren n tikta n Taliktrutiknikt, ideg yella awal n iliktrunik (ELECTRONIQUE) akked yisumuren-nni : Tamesliktrunt / taliktunit neɣ Timezzeft.
