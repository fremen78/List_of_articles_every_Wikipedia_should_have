Hosiftotta (mammals) (class Mammalia) kan jedhamani dhigaa-ho'a kan qaban, binensotta spinal-qabatotta ta'an fi kan itti bekamaan immo dubrii isaani anaan mammary gland kessatti burqisisuu fi akkasumas rifensaa, lafee gurra jiddu kan dhaga'uf tajajilamaan fi samuu isaani kessatti kutta neocortex jedhamu qabu. Hosiftonni bayeen ilkaan sirritii walitii qabame argamu kan qabani yoo ta'an akkasumas ontogeny kessatti placenta qabu. Samuun hosiftotta waan kessa (endothermic) fi circulatory system geggesaa, kunis kutaa afuraanni onnelle ni daratta. Hosiftonni ofii kessa gossa jiratotta (species) hanga 5,400 (namalle daratte), genera 1,200, family 153 fi order 29 ofii kessa qabu.
