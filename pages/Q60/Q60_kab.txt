New York d nettat ay d tamdint tameqqrant akk deg Yiwunak Yeddukklen. Tezga-d deg ugafa-agmuḍan (nord-est) n tmurt, ɣef Ugaraw Aṭlasi. Zedɣen-tt 8.336.697 n yimezdaɣen (19.006.798 s yegmamen ay d-yezzin fell-as). Tamdint-a tezga-d deg uwanak n New York.


== Isunen n weɣrem n New York ==
Llan semmus (5) n yisunen deg New York, s tenglizt qqaren-as Boroughs. 
Manhattan : agwans n temdint, tella af tegzirt n Manhattan. Llant deg-s Wall Street, Tasdawit n New York, Tasdawit n Columbia, asutel n Yiwunak Yeddukklen (ONU/NU). Ɣur-es imezdaɣen 1 546 856 deg-s.
The Bronx : ay d asun agellil akk deg weɣrem n New York akwd tdersiwin (minorités) tiberkanin d tispanikin maca tuɣal-d cwiṭ n teɣlist seg 20 iseggassen i yezrin. Ay d dduḥ n uẓawan Hip-Hop, deg-s illa Asarir Yankees Stadium. Ɣur-es  1 354 068 imezdaɣen deg-s.
Queens :  Ɣur-es  2 237 815 imezdaɣen deg-s. Yeḍra US Open dinna yal uggur n ctember, deg-s llan imafagen n JFK d La Guardia. Illa d agemim n tmezduɣt d tmugri (industry).
Brooklyn : Agemim n tmezduɣt n New-Yorkers. Ɣur-es 2 488 194 imezdaɣen deg-s
Staten Island :  Ɣur-es imezdaɣen 491 730 deg-s. Daɣem d agemim n tmezduɣt n New-Yorkers, yuggug si Manhattan.
