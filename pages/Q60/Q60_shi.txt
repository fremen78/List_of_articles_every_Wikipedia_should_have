Nyu Yurk (s tnglizt: New York)— ttussan bahra s ddu yism n Tamdint n Nyu Yurk (s tnglizt: New York City) fad ad yili umzaray d tsga n Nyu Yurk— nttat ad igan tamdint lli akkʷ ittuzdaɣn ɣ Iwankn Munnin, d nttat ad akkʷ imqqurn ɣ tsga n Nyu Yurk ula ḥtta ɣ Iwankn Munnin, tg ula tamdint nna iṭṭafn yiḍiṣ lli akkʷ ixatrn ɣ yigran n tsbbabt d iqaṛiḍn, asnɣms d tẓuṛi d tmlsiwin, tussna d ttiknulujya, asslmd d tmallayt.
Tga tamdint n Nyu Yurk yat ɣ wammasn n tsbbabt d iqaṛiḍn bahra istawhmman ɣ umaḍal, nttat ad igan tamaẓunt tadmsant n Iwankn Munnin acku ɣinn a ɣ munn timssurin d id bank imaḍlann, yili gis ula wammas n Timtta Imunn d Lbuṛṣa. Da fllas ttinin middn tamaẓunt n tadlsant d tadmsant n umaḍal.
Tbḍa tmdint n Nyu Yurk f smmust tmnaḍin: Manhaṭn, Brunks, Bruklin, Kwinz, Statn Ayland. Tettyussann s yan usuk ar as ttinin Asuk n Brudway, ar gis ttilin kigan n timzgunin, d Sanṭral Park yan ɣ tirtitin lli akkʷ ittyussann.
Ar ttaḍun iẓuṛan n tmdint n Nyu Yurk s usggʷas n 1624 lliɣ t bnan ad tg yan wammas ansbab s ifassn n kra n ihulanḍiyn, gan as yadlli s yism Amstrdam tamaynut g usggʷas n 1626. Tkka tt inn ddu tnbaḍt n ingliziyn ɣ usggʷas n 1664 sliɣ as gan yism n Nyu Yurk ḍaṛt ma ifka Tcarlz Wiss sin, agllid n Briṭanya, ikaliwn i gmas Duq Yurk. Tkka tt inn Nyu Yurk tga tamaẓunt n Iwankn Munnin zɣ 1785 ar 1790.
