Boros Sepanyol nopo nga iso boros mantad Boros latin. Ogumu tulun id pomogunan timpu baino momoguno boros diti gama soboogian mantad do bagas penjelajahan om penjajahan tulun Sepanyol di laid. Nokoponuridong yolo ogumu pogun wagu om minonimpuun do kopomorintaan wagu id pogun di laid no noturidong.
Id siriba nopo nga piipiro poomitanan do kinoyonon i momoguno boros Sepanyol sabaagi boros oponsol:
Id Koibutan Amerika, Amerika Tengah, om id Kepulauan Karibia: Costa Rica, Cuba, Republik Dominika, Guatemala, Honduras, Mexico, Nicaragua, Panama, Puerto Rico, om El Salvador. Sundung do lobi ogumu tulun momoguno Boros Inggeris id suang pilumaagan id Amerika Syarikat, nga nosiliu o Boros Sepanyol sabaagi boros kumoduo di bobos ogumu momoguno id pogun diti.
