Kilau nopo nga iso kawo do songongi sosial id suang do paganakan Formicidae, om poinsuang miampai moningot id nuludan Hymenoptera. Berevolusi kilau mantad sakag miagal moningot id zaman pintangaan timpu Kapur mantad 110 gisom 130 juta toun di nakatalib om nokoimbulai mogisusuai kawo kalapas kinoimbulaion susumuni kiwusak. Babaino, haro 12,500 lobi spesies dii notumbayaan do haro soginumu 22,000 spesies. Ouhan mongintutun kilau mantad antena om struktur miagal nod di kibontuk sondii miagal tawak di orikos. 
Kolobi do 13,800 mantad 22,000 spesies di kikiroon nimaan no pinosuai. Okito iti id nonggo nopo tindal id poinggonop winoun, suai ko' Antartika om piipiro pulau.
