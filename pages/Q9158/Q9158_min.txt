Surek elektronik (akronim: ratel, ratron, surel, atau surat-e) atau pos elektronik (akronim: pos-el) atau imel (bahaso Inggris: e-mail) adolah sarana kirim mangirim surek malalui jalur jaringan komputer (misalnyo Internet).

Dek surek biaso umumnyo pangirim paralu membayia per pangiriman (dek mambali prangko), tatapi surek elektronik umumnyo harago nan dikaluaan adolah harago untuak membayia sambungan internet. Tapi ado pangacualian misalnyo surek elektronik ka talepon genggam, kadang pembayiannyo ditagih per pangiriman.


== Sajarah ==
Surek elektronik alah mulai dipakai pado tahun 1960-an. Pado itu internet alun tabantuak, nan ado hanyolah kumpuan 'mainframe' nan tabantuak sabagai jaringan. Mulai tahun 1980-an, surek elektronik alah biso dinikmati dek khalayak umum. Sakarang ko banyak parusahaan pos di babagai nagari nan manurun panghasilannyo disebabkan masyarakaik alah indak mamakai jasa pos liak.


=== Surel Patamo Di Dunio ===
Konsep surel patamo kali dikamukakan dek Ray Tomlinson pado akhir tahun 1971. Ray Tomlinson pado wakatu tu bakarajo untuak Bolt Beranek and Newman "BBN" milik lembaga pertahanan Amerika. Pado awalnyo Ray baeksperimen dek program komputer nan banamo SNDMSG nan pado wakatu tu digunoan untuak maninggalkan pasan pado komputer, sahingga urang nan mamakai komputer dapek mambaco pasan nan ditinggaan dek pamakai komputer "wakatu tu normal katiko banyak urang manggunoan komputer nan samo sacaro bagantian".
Eksperimen Ray kamudian dilanjuikan dek manghubungkan SNDMSG malalui file protocol nan banamo CYPNET sahinggo program SNDMSG dapek mangiriman pasan ka komputer lain, namun masih ado dalam ciek jaringan ARPAnet. Eksperimen tasabuik barasil jo Itulah awal taciptonyo surel.
Pasan surel patamo kali nan dikirim dek Ray pado 1971 marupokan surel patamo di dunio. Isi Pasan barupo "kato kato tes" sapanuhnyo dilupoan, Kamungkinan pasan patamo nan dikirimnyo adolah QWERTYUIOP atau huruf sarupo," kato Ray dalam akun Flickr miliknya. Pado tahun 1972, Ray manganaan lambang @ "Indonesia dibaco ET" sabagai identitas email untuak mamisahan antaro Identitas pangguno dek domain. @ "bahasa inggris ditulih AT" aratinyo adolah "kapado".

== Metode pangiriman ==
Untuak mangirim surek elektronik kito mamaralukan suatu program mail-client. Surek elektronik nan kito kirim akan malalui babarapo poin sabalum sampai di tujuan. Mulai dari surek elektronik dikirim → Internet → POP3 server panyadio surel panarimo → mail client (di komputer si panarimo) → surek elektronik dibaco si panarimo
Nampak surek elektronik nan takirim hanyo malalui 5 poin (salain komputer pangirim jo panarimo). Sabananyo labiah dari itu sebab salapeh surek elektronik maninggaan POP3 Server mako itu akan malalui banyak server-server lainnyo. Indak tatutuik kamungkinan surek elektronik nan kito kirim disadap urang lain. Mako dari itu bilo surek elektronik nan kito kirim manganduang isi nan sensitif sabaiknyo kito mangarajoan tindakan pancegahan, dek mangacak (enkripsi) data dalam surek elektronik tasabuik (contohnyo manggunoan PGP, sertifikat digital, jo lain-lain)


== Caro mambaco surek elektronik ==
Surek elektronik pado awalnyo disimpan di dalam sabuah mailserver. Biasonyo katiko sasaurang mamakai koneksi ISP untuak sambungan ka internet, inyo akan dibarian ciek surek elektronik gratis. Surek elektronik nan ditarimo akan disimpan di server surek elektronik ISP.
Ado duo caro untuak mengakses surat elektronik:

Dek caro manggunoan paramban web, sarupo Mozilla Firefox. Metode iko disabuik sabagai web-based, aratinyo kito manggunoan media web sabagai parantaro ka kotak surek elektronik. Contoh: Yahoo! Mail dan Gmail. Untuak manggunoannyo, pangguna haruslah dalam keadaan daring. Layanan surek elektronik babasis web biasonyo disadioan dek penyelenggara layanan surel gratis.
Manggunoan program pengakses surek elektronik (e-mail client), sarupo: Eudora Mail, Outlook Express, Windows Mail, Mozilla Thunderbird, Mutt. Dek manggunoan program sarupo iko, sasaurang harus mengetahui konfigurasi nan biso didapek dari ISP. Keuntungannyo adolah dapek mambaco surek elektronik tanpa paralu tahubuang sacaro taruih-manaruih dek internet jo puluhan surek elektronik dapek ditarimo jo dikiriman sacaro basamo-samo sakaligus. Kalabihan nan lainnyo adolah perangkek luneik iko manyadioan fungsi-fungsi panyuntingan jo pembacoan surel sacara luring. Dek demikian biaya koneksi ka internet dapek dihimeik.

Jan mangirim lanjuik (forward) surek elektronik tanpa bapikia kagunoan bagi urang nan dituju.
Acok isi kolom subjek, jan dipadian kosong.
Dalam mangutip tulihan urang lain, acok usahoan mangutip saparalunyo, jan namua gampangnyo mangutip sado tulihan urang tu:
dalam manjawek surek elektronik urang lain, kutip bagian nan kito tanggapi se, salain labiah jaleh juo indak mamakan wakatu / jatah akses panarimo,

== Keamanan ==
Keamanan data di surek elektronik indaklah tajamin jo acok ado risiko tabukak untuak umum, dalam aratian sadolah isinyo dapek dibaco dek urang lain. Hal iko disebabkan dek karano surek elektronik tu akan malewati banyak server sabalun sampai di tujuan. Indak tatutuik kamungkinan ado urang nang manyadap surek elektronik nan dikiriman tasabuik.
Surek elektronik dapek diamankan jo mangarajoan teknik pengacakan (enkripsi). Salah satu program enkripsi nan populer adolah PGP (Pretty Good Privacy). Dek mamakai PGP mako isi akan dienkrip, jo hanyo urang nan tatuju dapek mandekripsi jo mambaco surek elektronik tasabuik. Karugiannyo adolah mambuek repot pihak pangirim jo panarimo (karano kaduonyo harus punyo program PGP, jo pangirim jugo harus punyo kunci umum panarimo, jo mangarajoan enkripsi pasan jo kunci tasabuik).
