Indunisya, s wudm amaddud Tagduda n Indunisya, tga yat tmazirt illan ɣ ugmmuḍ anẓulan n Asya d Iqyanusya. Tga arkabil akkʷ mqqurn ɣ umaḍal, ilint dars 17,504 tgzirin. Tada gitsnt akkʷ istawhmman gant tnt Jafa, Bali, Kalimantan, Sulawsi d Sumatra. Jakaṛta da illan ɣ tgzirt n Jafa ad igan tamaẓunt n Indunisya, maccan ttrsn kra n iɣawasn fad ad tt smmattin s Nusatara. Juku Widudu ad igan anssixf nns ɣil. Tbda Indunisya tatrart ɣ 17 n Ɣuct 1945. Ɣ 10 n tufat n wass n ljamɛ ann, iɣra Suykarnu d Hatta tasguri n tsimant n Indunisya. Iga wass n tsimant n Indunisya tigga tanamrut.
Tga Indunisya tis kkuẓt ɣ umaḍal ɣ umḍan n imzdaɣn nns nna ilkmn 238,452,952 n umddnu (aḍkaḍ n 2004), azgn nns izdɣ ɣ Jafa. Llan gis 111 ɣ ku km². Tlkm tjumma n wakal nns 1904000 km². Bahasa Indunisya ad igan tutlayt tamaddudt tanamurt n Indunisya, macc llant 749 tutlayin yaḍnin nna ittusawaln ɣ mnnawt tsggʷin n Indunisya zun d Tajafant, Tablint, Tasundant. Usṭralya ad igan taddjart n Indunisya zɣ unẓul, Sinɣafura zɣ ugafa utrim d Ufilibbin zɣ ugafa agmuḍan. Timizar yaḍnin aẓnin gant tnt, Babwa Ginya Tamayunt, Malizya, Timur Tagmuḍant da akkʷ idran iwtta f wakal d Indunisya. 
Amata n imzdaɣn nns gan imuslmn, macc ur tgi Indunisya tamazirt tanslmt s uslgn. Amḍan n imzdaɣn inslmn akkʷ imqqurn ɣ umaḍal illa ɣ Indunisya. Gr isgdan nna ḍffṛn ayt Indunisya, tlla Tmasiḥit (Tabrutistant d Tkatulikt), Tahindust, Tabudit d Tkunfusit).


== Tasgiwin ==

Tbḍa Indunisya f 34 tasgiwin. 5 zgitsnt llant yan waddad imẓli. Ku tasga tla tanbaḍt tuzrift d unbbaḍ nnsnt.
