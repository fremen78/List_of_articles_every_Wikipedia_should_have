Sinapan masin adolah sanjato api nan tatanam maupun portabel, nan mamiliki kamampuan manembak sacaro full-otomatis. Sinapan masin biasonyo manembakkan piluru sinapan nan diisi dari sabuak amunisi atau magazen bakapasitas gadang. Sinapan masin biaso manembak babarapo ratuih piluru per minik.
Istilah sinapan masin acok juo dipakai dek media massa dan dek bidang hukum, untuak mandeskripsikan jinih sanjato api apapun nan dapek manembak sacaro otomatis.
Sanjato api otomatis sarupo bakaliber nan labiah gadang dari 20 mm (0,79 in) diklasifikasikan sabagai mariam otomatis, indak sinapan masin.
