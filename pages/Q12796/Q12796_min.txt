Sanjato api (bahasa Inggris: firearm) adolah sanjato nan malapaskan ciek atau labiah proyektil nan didorong dangan kacapatan tenggi oleh gas nan dihasilkan oleh pambakaran suatu propelan. Proses pambakaran copek iko sacaro teknis disebuik deflagrasi. Sanjato api dulu umum manggunakan bubuk hitam sabagai propelan, sadangkan sanjato api modern kini manggunakan bubuk tak barasok, cordite, atau propelan lainnyo. Kabanyakan sanjato api modern manggunakan laras malingkar untuak mamberi efek putaran pado proyektil untuak manambah kastabilan lintasan.
