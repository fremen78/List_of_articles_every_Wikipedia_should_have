Jozef Visarionovič Stalin (s tarusit : Ио́сиф Виссарио́нович Ста́лин, s tajyurjit : იოსებ ბესარიონის ძე სტალინი), illul ass n 18 duǧember 1878 deg tamdint n Gori di Tamnekda tarusit (di Jurjya n tura), yemut ass n 5 meɣres 1953 deg Musku, d argaz uwanek seg imeqqranen asuvyati, d yiwen seg iwudam imdergen n umezruy n lqern wis 20, d akerwa ameqqran maḍi n Tadukli tasuvyitit, d azamul n umennuɣ n ugdud asuvyati mgal Tanaẓit.
