Rinanzat (Boros Inggilis: racism toi racialism) nopo nga' pogipiadan (segregation), pihipiad (discrimination), om niimpa' (prejudice) kumaa tulun, poingimpou id ranzat toi ko' tinaru diolo. Rinanzat nopo nga' iso ponoou (ideology) nogi, om nunu nopo pomomonsoi poinhakup do karaatan kumaa isaisai nopo tu' kointutunan ranzat om tinaru diolo oroitan do orinanzat. It gumayud toi ko' tutumanud do rinanzat nopo roitan do garanzat (racist). 


== Tadon hogot om komoon ==
Hogot do rinanzat mantad gamut ranzat do Boros Rungus,  kikomoyon mantad om race id Boros Inggilis. Milo nogi pinokomoi o rinanzat do koponunsungan do piniagat ranzat id voos do piniagat kosiwatan toi piniagat kootusan do mogikakawo tinau. Kawo do piniagat ranzat di tohuri nokomoi roitan nogi do piniagat dounkotos (substantive equality). Iso kawo tagayo do rinanzat nopo nga' rinanzat duridong (institutional racism) ii banis do iso pogun nopo mamasi do rinanzat om mongunsub wayataraat kumaa ranzat poingotul. Poomitanan, tongo tilin Jim Crow (Jim Crow laws) id Mirika Poinsompuu do mongotung tulun di ongoitom. Okon nopo id pisuk do banis poingiau o rinanzat nga' apasi nogi id garaid (bias) tadau adau. Kawo do rinanzat diti roitan do rinanzat mangalai (aversive racism). Kapangalayan nopo poingigom intaman (stereotype) om inaga'on suai di au otopot om paparaat do kouyuyuo' iso ranzat.


== Id Molisia ==

Id Pisompuruan Molisia, haro kawo do rinanzat duridong maya' do pihipiad danan (positive discrimination) id kapamansayan do toodo pongokun (affirmative action) montok tongoulun sandad roitan do Bumiputera (kikomoyon do Haing Tana'). Sawiawi Haing Tana' kikotunudan poimbida' poinggait id banis pisompuruan (federal law) om hinatan (ordinance) do damung (constituent states) lobi po id tongo pogun Sabah om Sarawak do kihinatan suai mantad id Malaya id pisuk do momogompi kotunudan ranzat sandad.  
Maya ponoriukan om totod (interview) di Dr. Trixie Tangit, id vantai (thesis) dau haro pisunsungan do kotunudan tinaru sandad Mamasok Sabah miampai do siompokon (agenda) di kosogindayan Tabai tu' okito niimpa' kumaa Momogun ii kogumuan okon Silam, tantad koimpuunon do Sabah poinsuang id pisompuruan Molisia. Poomitanan nopo nagarit id linapik toi ko' buuk kiuhu do The Golden Son of the Kadazan di Peter Mojuntin, hinombo do nakansang ngai Tatapa' (toi paado) om Tataka' (toi sisto) don Katolik.
