Ek game koi chij hae jiske log manoranjan (fun) ke khatir kare hae, aur kabhi kabhi iske parrhai me bhi kaam me laawa jaae hae. Ii kaam nai hae lekin kuchh game jiske professional khilaarri khele hae ke kaam bola jaae sake hae.
Game ke akele, team me, internet pe, paisa ke chaahe khaatir, chaahe nai, khela jaae hae. Ek game ke chaahe bina dekhe waala log ke aage aur chaahe dher log ke aage khelaa jaae hae. Jaise ke Satranj ke champinship ke dher jane dekhe hae. Ek khelauna (toy) aur game me atntar ii hae ki khelauna khele me koi niyam nai rashe hae , jab ki game me niyam rahe hae.
France ke sociologist, Roger Caillois aapan book, Games and Men, me batais hae ki ek game me ii chhe chij rahe hae:

"free" - khele ke compulsory nai hae
"separate" - game sab jagha aur sab time nai hoe hae
"uncertain" - jon log game khele hae ii nai jaane hae ki kon jiiti aur kon haari
"non-productive" - game se koi ke amdani nai mile hae
"governed by rules" - game ke niyam rahe hae jon samaj ke niyam ke rakam nai rahe hae
"fictitious" - jon log game ke khele hae ii jaane hae ki ii asliyat nai hae.


== Kuchh rakam ke game ==


=== Party game ===

Party game uu time khela jaae hae jab log ek-duusre se informally mile hae aur game se party me log ek-duusre se mile sake aur manoranjan hoe sake Kuch party game hae;

baatchiit ke khel, jaise "I spy"
daaru piye waala game
anumaan lagae waala gae, jaise ki "charade"
gaana gaae waala game, jaise koi topic pe gaao


=== Tabletop game ===

Ii game ke table ke uppar nai to koi flat jagha pe khelaa jaae hae, jaise ki board games, card games, dice games, miniature wargames, nai to tile-based games.
Board game me pieces ke kaam me laawa jaae hae. Ii piece ke board pe pahile se mark karaa gais game board me move karaa jaae hae. Kuchh board game hae draughts, chess, scrabble, backgammon.
Card game ke playing cards se khelaa jaae hae poker, solitaire, go fish, 500, cheat.
Dice game me ek, nai to ek se jaada dice ke roll kar ke aage waala move karaa jaae hae. Kuchh dice game hae  boggle, craps, ludo, yahtzee

Miniature wargaming ek rakam ke larrai waala game hae jisme sena ke hissa ke chhota chhota model se ek model battlefield me khelaa jaae hae.
Tile based game me koi rakam ke tile ke kaam me laawa jaae hae. Ek tile game hae dominoes.
