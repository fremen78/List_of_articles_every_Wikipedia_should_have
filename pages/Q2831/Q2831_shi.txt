Maykl Juzif Djaksun ilul ɣ Ɣuct 1958 ɣ Gari (tawankt n Wandyana) immut assf n 25 Yunyu 2009 ɣ Lus Anjlus (tawankt n Kalifurnya) iga anaẓuṛ Amirikani, amsiys d umarir d unḍḍam, ar fllas ttinin "Agllid n Upup" acku ilkm ɣaylli jju ur ilkim kra n umarir yaḍni ɣ ubdar d tsusnt d tmsaɣt n tassfifin, tsul tzzigizt nns kkuẓt n tmrawin n isggʷasn. 


== Tabyugrafit ==
Iga wis sa ɣ yat twja n tza n wactaln, issnti zikk tazzigizt nns lliɣ dars tlla 11 n usggʷas mayad d trabbut n "Jakson Five"  (Jaksun Smmus) ur ixdim waḥdu t ar 1971, rad issufɣ kra n irirn macc ur rad as nit ṛẓmnt tiggura n umurs ar 1979 kud lli issufɣ tirirt n "Off the wall" ig anaẓuṛ izwarn lli yutn kkuẓ n Tup 10 n irirn s yan walbum. fadd s "Thrailler" lli rad ig amzwaru lli akkʷ inzan ɣ wakud nns s 45 n umlyun n tunɣilin ɣ ussutl n umaḍal, ilmma issufɣ "Bad' ɣ 1989 mqqar ur  ilkim uṭṭun n tunɣilin nzanin zun d' "Thrailler" waynni yut sul yan umurs war anggu. 
ɣ tizwiri n 90's, immatti nn s yan umttul mqqurn ila uggar n yifḍ n uhktar issaɣ as ism n "Neverland" izwwq t inn s imudar d tsurarin. izzri daɣ yan isrsl n tnfalin ɣ wudm nns d inzar nns, ula ɣ yilm nns, irur tn s kra n tmukrisin n yilm, ɣ usggʷas n 1993, yiwi tumla is ikrfs kra n ufrux, imal n usggʷas ann rad itahl d Mari Prisliy macc dɣya lkmn uluf. itahl daɣ ɣ 1996 Dibura Ruwi lli dis yurun sin wactaln Brans d Bariz, iṛẓm as ɣ 1999. 
Ɣ 25 Yunyu n usggʷas 2009, immut Maykl Djaksun ɣ Lus Anjlus lliɣ dars tlla 50 n usggʷas, kra n imalasn hlli zwar yan ulmuggar ira gis ad ig "win adwal" s ugdud nns, ar ɣilad ur ta bbin ɣ ma iga amntil da s immut.
