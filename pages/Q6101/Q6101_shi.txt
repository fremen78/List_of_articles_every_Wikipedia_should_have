Marku Bulu niɣ d Maku Fulu (S taṭalyanit: Marko Polo) iga yan usbbab d yan umzuzf sɣ Binizya da iddrn gr usggʷas 1254 d 1325, ittyawssan bahra s imuddutn nns s Wasya lliɣ izgr aɣaras n umṭṭalfu (lḥrir) d tirra n "Udlis n yimḍiṛn" da iẓli i uglam n yimuddutn nns, adlis ad yiws nit i Yirubbiyn ad akzn wiming yaggugn ssnn f tdlsa n tmizar ann n Ccinwa d Lhnd d Lfuṛṣ d kra n tsgiwin yaḍnin.


== Tudrt nns ==


=== Zikk tudrt nns ===
Idrus ma fllas ittyawssann, imil ntta ilul ɣ Binizya, imqqur gis, kud lli iskr smmus d mraw n isggʷasn rad imnaggar d Babas  Nikulu d ɛmmis Maffyu da d urrinin sɣ umuddu nnsn, mnaggarn nit d tamttant n ubaba Klimun wis kkuẓ, saggmn aylliɣ zrin istayn n ubaba amaynu, d ɣ usggʷas 1271 umẓn daɣ aɣaras s tmnukda n imnɣuln d tiklit ad imun ditsn ufrux ann Marku dar llan ɣ wakud ann sa d mraw n isggʷasn. 


=== Amuddu ===

Kṛaḍ irgazn ann ad gan ɣil ad rad zgrn iẓṛayl d Iṛan, fad ad lkmn timizar n Wasya tanammast, kkn tiniri n Gubi d tɣrmt n Kambaluc ɣ rad kcmn d usarag n umnkud amnɣul Kubilay Xan igan yus n Jinkiz Xan, d da ibbin d tniɣitin da skrn imzwura nns, issusm amnkud uwway adlsan n ayt Binizya ad, Marku Bulu ilmd imikk s imikk tacinwit, ig akkʷ yan umsqad d umnbaḍ n tɣrmt n Yang Cu ɣ kṛaḍ isggʷasn, is ka ntta d babas d ɛmmis rad tn yaɣ umarg n Binizya, ttrn i ugllid Kubilay ad tn yajj ad aḍun, ɣmk ann nit, yuzn ditsn illis wan titmi acku ira ad tt isslkm s Iṛan, lliɣ zrin Sumatra d lhnd rad srsn tafruxt ɣilli tra, smdn ilmma aɣaras nnsn nttni s Ṭṭalyan, ɣ udwal nns s Binizya, rad imnaggar d yan imnɣi gr Binizya d Jinwa, ass n 8 cutambir 1298, Marku Fulu rad yaf agayyu nns ɣ ixf n tamaɣt n Kurzul, yat tamaɣt taylelant gr sin wafnatn ann, tirifit n Binizya rad tṛẓ d Marku Bulu rad ittɣʷi ɣ ifassn n Ijnwin, idru nn asqqur (abniq) nttan d yan umngal Ajnwi, Rustitcillu n Fiza, mi rad ittals tinfusin nns. Ɣmkad ad d ilul udlis nns "adlis n tibḍit n ddunit" yiwin sul ism n "udlis n imḍiṛn" . 
Mkann innuṛẓm Marku Fulu, yamẓ tidrfit nns, ikk udlis nns timizar akkʷ n Urubba, yaɣul s mnnawt tutlayin, acku ar gis ittglam sgud kullu ɣaylli iẓṛa ɣ umuddu nns. 


=== Tigira n wussan nns ===
Lliɣ ukan ismd 45 n usggʷas, iɣts Marku Fulu ad sar ur ikk kra n mani ɣ yill, itahl ilmma Dunata Badwir, yaru dis kṛaḍ n tarwa, immut nit dars ɣ Binizya ɣ wass n 8 Yinnayr 1324, ɣ yan ultay n sat tmrawin n isggʷasn.
