Marko Polo (s taṭalyanit : Marco Polo (asusru )), illul deg 1254 deg Vinisya, yemmut deg 8 yennayer 1324 deg Vinisya daɣen, d amzenzu, icaliyen ed amessikel avinisi. 


== Tudert ed yisiklen ==


=== Tawacult n Polo ===
Seg zik, tawacult n Polo teznuzay akked Asamar alemmas, terna ccan ed tabaɣurt imeqqranen. Deg 1260, tawacult-nni ruḥen seg Istambul ɣer timenaḍin n umalu n Tamnekda n imunguliyen, anda i teznuzay ɣef isɣunen. Syin akkin, tawacult n Polo ruḥen ɣer Buxara, ssufɣen-d assaɣen n tmidwa akked amenkad Qublay Xan, ed tettuɣal-d inmahalen-is deg Turuft.


=== Asikel ɣer Asya ===

Deg 1271, baba-s n Marko Nikkolò Polo ed ɛemmi-s Maffeo Polo d-uɣalen ɣer Tamnekda n imunguliyen, akked Marko Polo, yettεeddin ɣef Ɛakka, Yumurtalık, tamenaḍt n Xuṛasan (deg Iran n tura), Badaxcan (deg Afɣanistan n tura) ed timnaḍin nniḍen n Asya ِِ akken ad wwḍen ɣer Cangdu (tamanaɣt n imunguliyen) deg 1274-1275.
Azal n 16 neɣ 16 n iseggasen, Marko Polo ed tawacult-is ttidiren deg tzedduɣin n umenkad n imunguliyen. S lameṛ n umenkad Qublay Xan, Marko i yellan yelha-d d aṭas n tuɣdaḍin ɣer timnaḍin ibeεden n Tamnekda n imunguliyen.


=== Tuɣalin ɣer Vinisya ===
Deg 1290 neɣ 1292, tawacult n Polo ruḥen ɣer tamenaḍt tafarsit akked yiwet tageldunt n imunguliyen, syin akkin, i ruḥen ɣer Turuft, yettεeddin ɣef Istambul ed tagara Vinisya deg 1295.
Deffir tuɣalin-is ar Vinisya, Marko Polo ad yuɣal d ameḥbus n iǧenwiyen deg Ǧenova, deg lḥebs-nni Marko ittemlili-d ameḥbus n Pisa qqaren-as Rustichello d aneggal mucaɛen, Marko yebda yettaru ɣef 25 iseggasen-is deg Asya (adlis-nni qqaren-as s taṭalyanit : Il milione). Syin akkin, Marko yeffeɣ-d seg lḥebs ed yettuɣal ɣer Vinisya, anda yemmut  ɣer 70 n yiseggasen.


== Ayen yura ==


=== Il milione ===

Il milione d isem n ullis n usikel n Marko Polo, qqaren-as adlis n leεǧubat n umaḍal neɣ Weglam n umaḍal, Marko yura-t s tafranko-ṭalyanit d tutlayt tatrart i myussan deg leqrun 13 ed 14.
