Hulukust (s telmaneyt: Der Holocaust) (seg tegrikt: ὁλόκαυστος holókaustos,/ anda hólos anamek-is: yakk, kaustós anamek-is: rɣan) akken yettwassen daɣen s Šuah (si thibṛut: השואה, anamek-is: arwas) d tanɣaɣreft i d-yellan deg umsenɣi amaḍlan wis II tegla-d s tmenɣiwt n wazal n 6 n imelyan n wudayen n Uruppa sɣur Lalman tanazeyt d temsisiyin-is (timura n ugellus). Kra n yinmezruyen sseqdacen tabadut n Hulukust i wakken ad yegber ula d 5 n yimelyan niḍen n yimdanen ur nelli ara seg wudayen, yettwanɣen daɣen s yifassen n Yinaziyen i wakken ad yuɣal umiḍan ummid n teɣtas azal n 11 n imelyan n yimdanen. Tamhalt n tenɣi tella-d deg yakk ideggan n Lalman tanazeyt d tamiwin i d-tehres di Uṛuppa.
Anhil anazey d yimalalen-is gubṛen-d udayen, yerna nɣan-ten s tarrayt di tallit n umsenɣi amaḍlan wis II i d-yellan gar n 1941 d 1945, seg yiwet gar tenɣaɣrfin meqren yakk deg umezruy alsan. Asenter (aḥqar) d tmenɣiwt n wudayen yella d amur seg waṭas n usenter d tmenɣiwt s tarrayt i yexdem unagraw Hitler d imalalen-is mgal n waṭas n tegrawin tisertanin d kra n yiɣerfan aladɣa iziganen iṛumen d yikdawen (imeɛḍar) d inmeglayen isertanen d waṭas niḍen.
