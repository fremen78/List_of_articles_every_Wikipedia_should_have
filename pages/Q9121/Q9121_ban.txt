Atom inggih punika satunggil satuan dasar matéri, sané madaging antuk inti atom taler ambubu éléktron mamuatan négatip sané mangiterin ipun. Inti atom madaging proton sané mamuatan positip, lan néutron sané mamuatan nétral (tiosan ring inti atom Hidrogén-1, sané nénten ngelah néutron). Éléktron-éléktron ring satunggil atom kaiket ring inti atom olih gaya éléktromagnétik. Sekaa atom punika prasida mapaiketan saréng liyanan, miwah ngawentuk satunggil molekul. Atom sané madaging akéh proton miwah éléktron sané patéh magenah ring indik netral, sapunika sané madaging akéh proton miwah éléktron sané mabinayan masifat positip utawi négatip miwah kasengguh dados ion. Atom kakelompokang mawit akéh proton miwah néutron sané wénten ring inti atom punika. Akéh proton ring atom néntuang datu kimia atom punika, miwah akéh [néutron]] nentuang isotop datu punika.
Paistilah atom mawit saking Basa Yunani (ἄτομος/átomos, α-τεμνω), sané maartos nénten prasida kagetep utawi indik sané nénten prasida kakepah-kepah malih. Konsép atom dados komponén sané nénten prasida kawagi-wagi malih kapertama kanikayang olih para filsuf India miwah Yunani. Ring abad ka-17 miwah ka-18, para kimiawan magenahang aled-aled manah puniki antuk nganikayang pinaka datu-datu katentu nénten prasida kakepah langkung alit malih nganggén metode-metode kimia. Selami ungkur abad ka-19 miwah pamahbah abad ka-20, para fisikawan prasida ngacunduk struktur miwah komponén-komponén subatom ring jero atom, ngabuktiyang pinaka 'atom' nénten ja prasida kakepah-kepah malih. Prinsip-prinsip mékanika kuantum sané kaanggén para fisikawan salanturné maasil ngamodélang atom.
