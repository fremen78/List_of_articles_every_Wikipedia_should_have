Bikin (s tcinwat: 北京, " anamek-is tamaneɣt n ugafa ", Khan Baliq deg umezruy) d tamaneɣt n tegduda n Ccinwa, d tamdint tis snat meqqren deg Ccinwa deffir n Shanghai. Tamnaḍt n Bikin d yiwet seg rebεa n tɣiwanin tidelsanin deg tmurt . Agdud-is yewweḍ ɣer 14,580,000 n yimdanen deg useggas n 2003 AD - 7,5 imelyan deg temnaḍt taɣerfant tamnaḍt - dɣa d wis sin deg Ccinwa s yimezdaɣ deffir n Shanghai. Tamdint-a d tamnaḍt n tsertit, n tdelsa, d tussna d uselmed, tella d amkan n usikel n tmurt merra. Yiwen seg yismawen imeqranen n temdint-a, d Taddart Ixuṣṣen d Axxam n igenwan. D Beijing i d-yeṭṭfen amḍiq n wurar olampiyen n 2008 i meqqren akk deg umezruy n waddal.
