Tga Asya tamnẓawt akkʷ imqqurn ɣ umaḍal s tjumma d umḍan n imzdaɣn nns. Amata nns illa ɣ uzgn agafan n umaḍa. Tzdi Asya d Urubba ɣ utaram nns, tzdi d Ifriqya ɣ unẓul. Kra n tkkal ar smmunn mddn Asya d Urubba fad ad gint yat tmnẓawt imqqurn mi ttsmmun Urasya. Kra zɣ tɣrimiwin tinfganin akkʷ iqqburn ssntant ɣ Asya, s umdya Sumar, Ccinwa d Lhind. Tga Asya adɣar n tlalt n mnnaw isgdan (id ddin), s umdya, Tahindust, Tazradict, Tudayt, Tajaynt, Tubdist, Takunfuct, Taṭawt, Tamasiḥit, Lislam d Tsixit. Tg daɣ akal n tlalit n kra n imnkad mqqurnin wan Umnkad Afarsi, Amnkad Amuɣal, Amnkad Amungul, d Umnak n Ming. Tga tmnẓawt ad akal n 44 n tmizar s tdrsi.


== Tajumma ==
Tamu Asya kigan n ikalln, tamẓ 30% zɣ tjumma n wakal n umaḍal, izri umḍan n imzdaɣn nns win tmnẓawin yaḍnin s 60% n imzdaɣn n umaḍal nna gis illan.
