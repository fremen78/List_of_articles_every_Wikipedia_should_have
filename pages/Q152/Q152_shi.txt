Islman gan imudar izdɣen g waman, ar-ten ictta ufgan, llan tugt n wanawen n iselman neɣ n imudar izdɣen g waman.


== Anawn n islman : ==
Llan tugt n wanawen n iselman, ɣayan af raten nsemsasa g trubba d iwckan g izddar :


=== Id bu wqqcur ===
Iselman id bu-wwqqcur gan iselman mi tdel tafegga nsen s iqqcran :


=== Id war aqqcur ===
Iselman id war-aqqcur gan iselman m-ur tdil tafegga nsen s iqqcran :


=== Iwqqasen ===
Anaw-ad n iselman gis iselman d-a yrwasen Awqqas:


=== Imlwaɣen ===
Imlwaɣen  i-tt-yussan dar ingmaren (n ill) d isbbaben n iselman : 

Azayz
Amermeḍ
Mmu-tlusi
Imlwaɣen-ad irna-t-nn umarra n udlis "Sus d yill" : 

Abɣaynuz
Iḥya


=== Tizmkin ===
Tizmkin i-tt-yussan dar ingmaren (n ill) d isbbaben n iselman :

Abrar
Axucan
Ẓeɛḍuḍ
Tankurt
Gaga
