Tga Miksinu tamazunt n Lmiksik , tettyawbna  ɣ tasut tis 16, ɣ 2260m n tattuyt ssɣ iggi n yill, f isagurn n yan wansa aqbur lli mi ismn "Tinuctitlan" igan tamaẓunt n tgldit n waẓṭik, tiɣmt tga yan uwggay n iswak ibaṛukn d itrarn ula iḥcac (lbrark). 
Tiɣrmt llan gis uggar n 21 umlyun n imzdaɣn, tga tuẓẓumt n tdlsa d tdamsa n tmazirt n Lmksik. Tili gis yat tsnniɣt ittamun mad iggutn ɣ id tugga n tɣrmiwin lli zrinin zwar n ulkkum n Kulumbus.
