Jerman toi ko' ngaran nokoimagon Republik Persekutuan Jerman (Templat:Audio-de, IPA: [ˈbʊndəsʁepuˌbliːk ˈdɔʏtʃlant]), bopo nga iso pogun id Eropah Tengah. Pogun nopo diti nga id piolitan do Laut Utara, Denmark om Laut Baltik id koibutan; Poland om Republik Czech id kosilahon; Austria om Switzerland id kabaatan; om Perancis, Luxembourg, Belgium om Belanda id kotonobon. Ginayo nopo do pogun diti nga kiikiro 357,021 km² om kiwaa iklim osuhatan kimusim. Miampai lobi 82 riong mogigion, ginumu nopo diti nga bobos ogumu id suang Kesatuan Eropah suai ko tinimungan mondidikot sompomogunan kotolu bobos agayo.
Nosuat no o rekod kokomoi Germania, iso kinoyonon i ionon do tulun-tulun Jermanik pogulu 100 M. Mimpuun do abad ko-10, minombontuk o wilayah-wilayah Jerman do toruntud Empayar Suci Rom i nokotindohoi gisom toun 1806. Ontok abad ko-16, nosiliu o kinoyonon id koibutan Jerman do longkod koguraan Reformasi Protestan. Sabaagi negara bangsa zaman moden, pinosompuru kawagu ii ontok Sangod Perancis-Prusia ontok toun 1871. Kalapas Sangod Pomogunan II, numaan kawagu pitongkiado o pogun diti do kuasa-kuasa Barat. Ontok toun 1990, pinisompuru kawagu o Jerman Timur om Jerman Barat.
Jerman Barat nopo nga pogun i minonuridong do Komuniti Eropah ontok toun 1957, om sumusuhut nosiliu Kesatuan Eropah ontok toun 1993. Pogun nopo diti nga soboogian mantad koiyonon aiso piolitan Schengen om mimpuun momoguno mato tusin miampai Eropah, Euro ontok toun 1999. Jerman nopo nga iso republik persekutuan berparlimen ii kiwaa 16 buah pogun bersekutu. Ibu negara nopo nga Berlin om kakadayan bobos agayo nopo nga Frankfurt. Minamung nogi o Jerman do PBB, NATO, G8 om minanain do Protokol Kyoto. Ii nogi nga ekonomi kotolu bobos agayo id pomogunan tumanud KDNK om pengeksport kakamot padagangon bobos agayo ontok toun 2007. Tumanud nilai mutlak, mongobi o Jerman do kouhupan koburuon bobos agayo id pomogunan, om balanja soudor dau nopo nga id taang koonom. Nakalantoi o pogun diti popoburu koposion di akawas om minonuridong iso sistem koumoligan sosial miampai pointongkop. Mongigit ii tonggungan oponsol id suang ahal Eropah om popotindogoi piuhupan ogirot mogikaakawo taang id pomogunan. Suai ko ii, nookunan nogi o Jerman sabaagi momuruan id suang mogikaakawo gana sains om teknologi.


== Sajara ==


=== Puak-puak Jermanik ===

Tadon nopo kaaka dot tulun-tulun Jermanik nga mantad Zaman Gangsa toi ko id maso Zaman Bosi pogulu era Rom. Puak-puak diti nokoburu mantad id kaabatan, kosilaon om kotonobon do Jerman. Puak diti nakaanu nogi minomonsoi do pionitan labus miampai puak suai id Eropah Kotonobon miagal ko tulun-tulun iran, puak Balt om Slav miampai Puak Celt id gaul.
Maso pomorintahan di Augustus, songulun jeneral rom, Publius Quintilius Varus tuminimpuhun manganu kokuasaan id tanah Germania. Mantad pisangadan diti nakaanu manahak do puak germanik kokomoi koinganan taktik it naanu mantad tongoulun rom. 


=== Empayor Rom Tolundu (962-1806) ===

Empayor zaman pintanganan diti naasil mantad pibobogian do Empayar Carolingia id toun 843, it pinointutun di Charlemagne id tadauwulan 25 Disember 800 om nokoingkakat id mogisuai-suai bontuk gisom id toun 1806.


=== Pomulihan om Revolusi (1814–1871) ===

Mantad kinorotuan di Napoleon Bonaparte, nakaanu mititimburungak oo Kongres Vienna id toun 1814 om pinopoingkakat do Koisaan Jerman 
(Deutscher Bund), ii no tu iso liga it oluwau ii pinopoiso do 39 negeri-negeri berdaulat.
