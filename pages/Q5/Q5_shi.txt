Afgan (s tsrɣint : إنسان) iga yan wanaw abyujuji, ism nns amassan "Homo sapiens". Ɣ 2018, llan uggar n 7 imlyarn n mddn ɣ iggi n wakal. Taguri afgan ur dars amagut d ikkan aẓur nns nit, macc tlla tguri n umagut "middn".


== Izmaz iwtayn n ufgan ==
Arraw (nɣ Aḥclaf) : Afgan urta iluln.
Taḍṛant : 0 - 2 isggasn
Tazzant : 2 - 10 n usggas
Amawaḍ :10 - 18 n usggas
Abliɣ (nɣ Amtrug) : 18 - 60 n usggas
Awssar : uggar n 60 n usggas

		
			
			
		
		
			
			
		
		
			
			
		
		
			
			
		


== Anaw n ufgan ==
Llan sin wanawn n ufgan, argaz d tmaɣaṛt.

		
			
			
		
		
			
			
		


== Iẓuṛan n ufgan g umaḍal ==
Nẓẓḍaṛ a nini mas afgan n umaḍal dars kṛaḍ iẓuṛan : 

Aẓuṛ amanguli
Aẓuṛ agnaw
Aẓuṛ aquqazi

		
			
			
		
		
			
			
		
		
			
			
		


== Ẓṛ uggar ==
Tafgga n ufgan
