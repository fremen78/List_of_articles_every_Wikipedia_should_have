Mirkur nɣ Azwaq (akrraj: ) iga amtiwg lli kullu imẓẓiyn ɣ unagraw anafuk d uttayn ur iṭṭif awd kra n unzwi i ra yas iḥḍu tirɣi, ɣik ann a f ar nttafa tasna n uzwaq ar ttlmaway amzaray ɣ tskʷfal n tirɣi lli kullu imqqurn gr imtiwgn n unagraw ad.


== Asamu ==
Awum aknunnay: 4 879 akilumitr.
Abddur: 3,3 × 10²³ akilugram.
Astum f tfukt: 59,6 agndid akilumitr.
Azmz n wannaḍ: 59 n wass.
Azmz n uḥbu anafuk: 88 n wass
Aẓẓug n tsna: 3,7 M/S².
Tirɣi n tsna: 167 ʷC.
Tirɣi - azal: 473 ʷC.
Tirɣi - iḍ: -183 ʷC.


== Tamdayt ==
Ar ittnunnt uzwaq zun d kra n itri n zikk nɣd win tdggʷat ɣ wakal, macc icqqa bahra a ittuẓṛa acku yaẓ nn i tfukt, d waxxa ɣikan, nẓḍaṛ att nkid d att nanni snat tikkal ɣ usggʷas: ɣ tldrar (tafsut) dar timdllst d ɣ umwan dat n awjim.


== Tanɣa n uzwaq ==
Ifrki n uzwaq trwas ultmas ɣ wayyur nnɣ iɣ llan isaffn ..., imẓṛiyyn ittuẓlan i yifrki n tsna d gan t kigan n tiwririn tukmirin xttilli iẓḍaṛn mas yadli igan ul n uzwaq d taduli isiṣmiḍn d ar ittimẓi ffir n iɣ imyamaẓ ifrki.


== Wul ==
Tuqqna yattuyn n uzwaq inza mas d ul nns iga anbɣur s uzzal, uggar n ku imtiwgn wiyyaḍ ɣ unagraw nnɣ anafuk.
