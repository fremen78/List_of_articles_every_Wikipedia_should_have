Portugal toi ko' ngaran nokoimagon Republik Portugal (Portuguese: República Portuguesa) nopo nga iso pogun i kogumuan hilo id Semenanjung Iberia di Eropah. Portugal nopo nga pogun berdaulat bobos kotonobon id Eropah, miwolit miampai Sepanyol id koibutan om kosilahon om Lautan Atlantik id kabaatan om kotonobon. Koiyonon pmorintaan kaampai no Kepulauan Atlantik, Azores om Madeira, koduo-duo koiyonon autonomi miampai Watas tempatan sondii.

Portugal nopo nga pogun bobos otuo id Semenanjung Iberia, om iso mantad pogun otuo id Eropah om id pomogunan, miampai nosugkuan o koiyonon pomorintaan, om pinisangod montok om pinorinta mantad timpu prasejarah. Tinaru pra-Celt, Orang Iberia, tulun Celt, tulun Carthage om tulun Rom nosugkuan (diserang) do tinaru Visigoth om puak-puak Jermanik Suebi. Kalapas penaklukan Islam id Semenanjung Iberia, kogumuan koiyonon Portugal nopo nga soboogian mantad Al-Andalus montok piipiro abad. Portugal sabaagi pogun tinuridong ontok Reconquista Kristian di pogulu. Tinuridong ontok otun 868, nointutunan o Watas Portugal kalapas Pertempuran São Mamede ontok toun 1128. Pinoimagon o Kerajaan Beraja Portugal om sumusuhut tinumanud do Pertempuran Ourique ontok toun 1139, om kagabasan mantad Kerajaan León ointutunan miampai Perjanjian Zamora ontok toun 1143.


== Demografi ==

Statistics Portugal minongintob do pointonod 31 Momuhau 2022, ginumu mogigion Portugal nopo nga 10,467,366,id nombo 52.2% nopo nga tondu om 47.8% nopo nga kusai.  Ontok 2024, aaga' (jangkaan) koposion median Portugal nopo nga 82.8 toun. Mogigion Portugal nopo nga homogen, di gulu majoriti mogigion dau nopo nga tinaru Portugal i kiugama Islam. Nga kalapas kaantakan reconquista, mononsog o Karajaan Portugal do koinsanai mogigion dau do papawaliu kotumbayaan kumaa Rom Katolik.


== Pulitik ==

Portugal mongumboyo pogun dau sabaagi republik demokratik miampai kotoinaan semi-presidensial pointonod Ko'oturan 1976 angkab. Konstitusi pogun ilang ahod pulitik id suang organisasi kotoinaan Portugal kosili apat pongigit ginaras: Presiden Republik, Luguan Montiri, Dewan Pogun om Kuut.


== Sipoot ==

Buul sipak nopo nga sipoot bobos maran-ngaran id Portugal. Haro mogikaakawo kejohanan mantad kalas amatur gisom kaadangon sompomogunan. Mamamain baanan irad Eusèbio, Ricardo Quaresma, Rafael Léao om Cristiano Ronaldo nopo nga gonding apangkal id suang sajara buul sipak Portugal modern. Piipiro momomoluda Portugal abantug om kolimbou irad José Mourinho, Abel Fereirra om Luís Castro.
