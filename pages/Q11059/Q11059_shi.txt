Tasanskritt tga yat tutlayt Tahndit d Tanipalit ikcmn ɣ twja n tutlayin indu-raniyn lli igan s yixf nns yat talat ɣ twja imqqurn n tutlayin indu-ruppiyn da ismunen ula Tagrikit d Tṛmmuyt d Tjṛmanit. 

Tutlayt-ad tga bahra taqburt, tddr mnnaw n wafḍan n isggʷasn, tg ula yat tutlayt tafuẓaṛt ɣ usgd Ahindus, d ar ttɣaln is d Bṛahma ad tt iskrn isslmd tt i ayt wakal, ɣayann af as zdin ism n "Sam skrit" (Kra imuccdn),  tutlayt-ad tlla ḥtta ɣ usgd Abuday, d Usixiy, d Ujayni. Mqqar drusn ɣwilli srs sawalnin ɣikk-ad, waxxa ɣik-ann ar nit tettyawxdam ɣ tɣri n idlisn ifuẓarn. 
Kra n 24821 n yan ad sul isawaln tutlayt-ad.


== Askkil ==
Askkil akkʷ ittyawssann i tutlayt-ad iga Divanagari, lli sul igan win tirra n tutlayin n ugafa n Yindya. Askkil-ad icqqa i tirra ur d zun d win Yifiniqn, acku ur ar issxdam id llif, ar issxdam angraw n tfirin.
