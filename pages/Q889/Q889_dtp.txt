Afghanistan  /æfˈɡænᵻstæn/  (Pashto/Dari: افغانستان, Afġānistān), nopo nga iso pogun ii nosorili do tana ii poingiliu id suang Asia Kabaatan om Asia Tengah. Haro aga'-aga' 31 riong tulun, minoposiliu do pogun ko 42 id bobos ogumu mogigion id pomogunan. Pogun diti nopo nga id wolit do Pakistan id kabaatan om kosilahon; Iran id kotonobon; Turkmenistan, Uzbekistan om Tajikistan id koibutan; om China id kosilahon rahat tosodu. Bidak watas pogun diti nopo nga 652,000 km2 (252,000 bt2), minoposiliu do pogun diti ko-42 tagayo id pogun.
