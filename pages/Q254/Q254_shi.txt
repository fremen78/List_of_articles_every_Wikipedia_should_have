Bulfgang Amadyis Muzart ilul ɣ Salzburg yat tɣrmt illan ɣ Nnmsa (n ɣassad) ɣ wass n 27 Yinnayr usggʷas 1756, immut ɣ Binna ɣ wass 5 Dujambir n usggʷas 1791, Iga yan unḍḍam niɣ d mafmaway n uẓawan aklassik mi tkka tsillit nns ddunit akkʷ. 
Iga Amadyus yus n Lyubulḍ Muzart igan s iɣf nns yan uslmad n Kamanja d yan unḍḍam dar usarag n ugldun d uqdic n Salzburg, ittyawssan sɣ mẓẓi nns is iga tifrɣi n waḍu, lliɣ yad iskr ṣḍiṣ n isggʷasn issnti irimn nns zwarnin ɣ tmafmawayt (tawssna n tmawayt), ifrk babas i ufulls n iwis ɣayann af rad srs ikk mnnawt tmizar ɣ urubba, ɣ usggʷas  1787 rad immatti ɣ umggaru s Binna ɣilli ɣ rad iɣʷi assaɣ n Mafmaway n tgnart (aḥanu) n usarag agldan, yufa taldni (tamɣart) nns igan yat tmarirt mi ismn "Kunṣtantz Wibir" ɣ tɣrmt ann nit, immut Muzart zikk kiɣ ka dars 35 n usggʷas kiɣ ukan iswuruy f "Rukyim", asrag n tmttant nns iɣama yan usdɣas. 

Mqqar tudrt nns igzzuln inẓm Muzart ma ittggan 24 n ubbira, ad gisn ittyawbdar: "Ḍḍun Jyubani" d "Tgamut taḍlmaṣt" d "Tmɣra n Figaru" , issurf wuṭṭun n tmawayin nns smmust tmrawin yat, ikka srsnt anawn d unsawn akkʷ n uẓawan, zun d ubbira d samfuni d kunsirtus d sunatas imk d imk, ifl ilmma tisaslin nns ɣ umaḍal ad ... 
Illa ma ittinin is d Ntta d Bax d Bithufn gan imɣarn niɣ d imayṣṭṛutn n uẓawan Aklassik id win akkʷ izmaz.
