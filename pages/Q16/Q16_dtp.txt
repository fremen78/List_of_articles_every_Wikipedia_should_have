Kanada (English: Canada, Koroitan:  ) nopo nga iso pogun do Koibutan Amerika i kaampai no do hopod provinsi om tolu wilayah. Poinladsong id boogian koibutan bonua dau, dumimpot o Kanada mantad Rahat Atlantik id kosilahon gisom Rahat Pasifik id kotonobon om Rahat Artik id koibutan. Kanada nopo nga pogun koduo bobos agayo id pomogunan. Pialatan Kanada miampai do Amerika Syarikat id kabaatan om koibutan kotonobon nopo nga i bobos anaru id pomogunan.


== Nontodonon do ngaran ==
Ngaran Canada data dau nopo nga mantad boros Iroquois St. Lawrence, ii nopo nga hogot kanata i kirati do "kampung" toi ko' "pogigianan". Ontok toun 1535, mogigion sandad id rantau i dinondo nosiliu do Kakadayan Quebec mamarait hogot kanata soira popotuduk ralan kumaa kampung Stadacona kumaa mononompogunan Perancis, Jacques Cartier. Sumusuhut minomoguno i Cartier do hogot Canada okon ko' nopo popokomoyon do kampung dii nga pointongkop nogi do kinoyonon taklukan Donnacona (luguan Stadacona); miririkot toun 1545, tumimpuun o buuk-buuk om peta-peta Eropah do momolohou rantau dii miampai ngaran Canada.
Ontok abad ko-17 om osopung do abad ko-18, Canada nopo nga ngaran montok soboogian wilayah Perancis Wagu i poinladsong id poingompus do Bawang Saint Lawrence om kotogisan koibutan do Tasik-Tasik Agayo. kinoyonon dii nabagi kumaa duo jajahan British, Upper Canada om Lower Canada. Pinoiso di kawagu goduduo dii sumiliu do Wilayah Kanada (Province of Canada) ontok toun 1841. Kalapas nokosiliu do Pogun Piomungan ontok toun 1867, ngaran Canada notorimo sabaagi ngaran rasmi pogun wagu diti, om Dominion (i naanu mantad Mazmur 72:8) tinahak sabaagi pongundadait pogun. Mantad dii, gunoon o ngaran poimponu Dominion of Canada do sangangadau gisom 1970-an. Soira minongintalang o Kanada do autonomi politik dau mantad do United Kingdom, asaru nodi gunoon do karajaan persekutuan o Canada nopo id dokumen karajaan om pibatasan; kinosimbanan diti pinointalang maya kaalanan ngaran koundarangan tadau kabansaan mantad Tadau Dominion kumaa Tadau Kanada ontok toun 1982.


== Sajara ==


=== Tulun sandad ===
Monokodung o ponoriukan-ponoriukan arkeologi om sajara genetik tulun sandad Amerika do haro o tulun id koibutan Yukon ontok 26,500 toun di pogulu om id kabaatan Ontario ontok 9,500 toun di pogulu. Old Crow Flats om Bluefish Caves nopo nga duo poomitanan do koiyonon-koiyonon arkeologi bobos totuo montok kinoyonon tulun Paleo-Indian id Kanada. Id taatanga' do tulun sandad i ointutunan sabaagi Tinaru-Tinaru Kumoiso, haro walu susuyan unik kokomoi do pomorunan pomogunan om adaptasi-adaptasi dau. Woyo-woyo tulun Pribumi Kanada nopo nga kinoyonon di pointimporon, pomutanaman, inus wonsoi awam om istiadat, kompleks hierarki tulun ginumuan om rangkaian pomutaranan. Soboogian mantad do koubasanan diti nopo nga alaid no natalup maamaso kinorikatan do Eropah di kumoiso (kk. kolimpupuson abad ko-15-kotimpuunon abad ko-16), om nokito kawagu maya ponoriukan arkeologi.
Mogigion tulun sandad nopo nga pokiikiro do 200,000 tulun om duo riong ontok abad ko-15,miampai 500,000 nopo nga notorimo no do Suruhanjaya Diraja Kolidasan Pribumi Kanada.Kinaantakan wabak rumoliu di mingguli i noowit do mondidikot Eropah miagal ko' lumasu om longohon, campak, om cacar, om nogi bagas-bagas do suai mantad do piromutan miampai tulun Eropah, minongingkuri do ginumu mogigion sandad 40-80% kalapas do piromutan dii. Tulun sandad id Kanada nopo nga kaampai no Tinaru-Tinaru Kumoiso, Inuit, om Métis. Koubasanan do kacukan Métis nopo nga mantad do pintangaan abad ko-17 soira suminawo o tulun Tinaru Kumoiso om Inuit miampai do mondidikot Eropah. Amu kopio miromut o Inuit miampai mondidikot Eropah ontok do timpu panaladan.


=== Penjajahan Eropah ===
Nokorikot o tulun Eropah di timpuunon nogi soira minogigion i pelaut Norse id L'Anse aux Meadows, Newfoundland do oruhai ontok pokiikiro toun 1000. Kalapas dii, au no ruminikot kawagu o tulun Eropah gisom toun 1497, soira minonoroka o songulun pelaut Itali i pinungaranan do John Cabot (Giovanni Caboto) do kotogisan Atlantik Kanada montok do England. Ontok toun 1498 gisom 1521, minonuridong i pelaut Basque om Portugis do kota kara pongintorusan do sada paus om ponginsadaan kitimpu poingompus do kotogisan Atlantik. Ontok toun 1534, tuminongkop i Jacques Cartier do Bawang Saint Lawrence montok Perancis. Mononompogunan Perancis,Samuel de Champlain nokorikot ontok toun 1603 om minonuridong do pogigionon Eropah di pointimporon i kumoiso id Port Royal ontok toun 1605 om Kakadayan Quebec ontok toun 1608. Monoiso o Canadiens kinoyonon doros do Bawang Saint Lawrence om Acadiens nopo nga poingion id kinoyonon do Maritimes, Papapataran do hulu Perancis om mubaligh Katolik nopo nga kuminaa id Tasik-Tasik Tagayo, Teluk Hudson, legeh Mississippi gisom id Louisiana (Perancis Wagu). Sangod Beaver nopo nga sabap no do piahawan monguasa' do pomutaranan hulu Koibutan Amerika.
Pulau St. John (dinondo Pulau Prince Edward) nokosiliu do jajahan di misuai ontok toun 1769. Montok mongilag do konflik id Quebec, pinolulus do British o Akta Quebec ontok toun 1774 montok papalaab wilayah Quebec gisom id Tasik-Tasik Tagayo om Bawang Ohio, nga popotimporon do boros Perancis, kotumbayaan Katolik, om kooturan sivil Perancis hilo. Ahal diti nokotogod do ogumu mogigion id Hopod om Tolu Jajahan om minonguhup do kinoimbulayan Revolusi Amerika.
Pibatasan Paris (1783) minangakun do kagabasan Amerika Syarikat om minanahak wilayah-wilayah id kabaatan Tasik-Tasik Tagayo kumaa Amerika Syarikat. Soginumu 50,000 tulun Setiawan Empayar Bersatu minongiduan do Amerika Syarikat om kuminaa id Kanada. Pinotongkiad o New Brunswick mantad do Nova Scotia id suang ponguludan kawagu pogigianan Setiawan id kinoyonon Maritimes. Montok monokodung do puak Setiawan i moboros do Inggiris id Quebec, pinabagi do Akta Perlembagaan 1791 o wilayah di kumaa Lower Canada i moboros do Perancis om Upper Canada i moboros do Inggiris, om sumusuhut minanahak tikid wilayah dii iso Dewan Undangan.

Angkabon do mombontuk karajaan i kitonggungan minogowit do kinotingkodon do Pemberontakan 1837. Kalapas dii, pinoposogu o Ruputan Durham do karajaan kitonggungan om nogi asimilasi Kanada Perancis kumaa koubasanan British. Pinaamung do Akta Kesatuan 1840 o goduduo Kanada dii sumiliu do Wilayah Kanada di miiso. Pinoturidong nogi o karajaan kitonggungan montok oinsanan wilayah Koibutan Amerika British gisom toun 1849. Mantad do ponoinan Pibatasan Oregon do Britain om Amerika Syarikat ontok toun 1846, nopupusan o pioduhan pialatan Orgeon, om pinapalaab do pialatan id ponong do kotonobon poingompus garis lintang 49 darjah Koibutan. Iti nopo nga minongukab wayaan kumaa ponuridangan do jajahan-jajahan British id Jajahan Pulau Vancouver om British Columbia.


=== Timpu moden ===

Minampayat do Kanada o Newfoundland (dinondo Newfoundland and Labrador) ontok toun 1949. Kosunion do ekonomi Kanada pasca-sangod om nogi dasar-dasar karajaan Liberal di mogisusuhut, pinapaharo iso identiti Kanada i wagu, i natanda' maya ponorimoon do Bondira Kanada ontok toun 1965, kopoindalanan dasar dwiboros rasmi (Boros Inggiris om Perancis) ontok toun 1969, kopogisusuaian do koubasanan rasmi ontok toun 1971. Ponuridangan ababayan-ababayan demokrasi sosial miagal do panamangan kolidasan sajagat, Palan Pinsin Kanada, om Kaalasan Siswa Kanada, sundung do piipiro karajaan provinsi, loolobi no Quebec om Alberta, amu osokodung do dasar-dasar diti tu sabaagi koligogon kumaa bidang kuasa diolo. Tohuri no, panangbarasan parlambagaan i pinaharo do mogisuhut-suhut minogowit kumaa "penyetempatan" parlambagaan Kanada mantad United Kingdom ontok toun 1982, i miampot miampai pombontukan do Piagam Hak om Kosinggawaan. Ontok toun 1999, nosiliu o Nunavut do wilayah kotolu do Kanada kalapas alaid no do minogibooboros miampai karajaan persekutuan.
Nokotoguang o Quebec do kosimbanan sosial om ekonomi di aralom maya Revolusi Opian ontok 1960-an i pinapaharo do gura nasionalis moden. Parti radikal Front de libération du Québec (FLQ) pinopoimbulai do Krisis Gumas ontok toun 1970. Pinoingkakat o Parti mononokodung kedaulatan Parti Québécois montok mombontuk karajaan ontok toun 1976, om kalapas dii pinapaharo do pomuhuan rolou kokomoi do Koguraan kedaulatan Quebec i amu minongobi do angkabon dau ontok toun 1980.  Pongumbalan montok monokodung do gelombang nasionalis Quebec miampai parlambagaan maya Persetiaan Meech Lake ontok toun 1990 nga amu nogi nakalantoi. Ahal diti minogowit kumaa pombontukan Bloc Québécois id Quebec om nogi kalangkason do Parti Ponginwoguan Kanada id Kotonobon Kanada. Pinaharo kawagu iso pomuhuan rolou ontok toun 1995, hinonggo amu notorimo o kedaulatan miampai majoriti tipis 50.6% kumaa 49.4%. Ontok toun 1997, pinootus do Mahkamah Agung Kanada o pitongkiadan unilateral do iso provinsi di amu tumanud parlambagaan, om pinolulus do parlimen o Akta Kejelasan montok mamagarit do syarat-syarat pitongkiadan terunding mantad do Piomungan.
Suai ko' mantad do kobolingkaangan kedaulatan Quebec, nokotigowo nogi o tulun Kanada do piipiro kobolingkaangan ontok kotohurionon do 1980-an om kotimpuunon 1990-an. Kobolingkaangan dii kaampai no kinoloputon do Kotuludon Air India 182 ontok toun 1985, pamatayan di ogumu i bobos do araat id suang sajara Kanada; pamatayan École Polytechnique di ogumu ontok toun 1989, ii nopo nga kinaantakan mogititimbak i moniru' do mokiikinobos; om Krisis Oka ontok toun 1990, konfrontasi orongit di bobos osopung do karajaan om nogi puak Pribumi. Minaganu nogi o Kanada do bbogian id suang angkatan miamung linuguan do AS id Sangod Teluk ontok toun 1990, om minimparagat nogi id suang misi-misi ponoronongon ontok kolimpupuson do 1990-an. Pinapaatod o Kanada do soudor kumaa id Afghanistan ontok toun 2001, nga amu minangatod soudor kumaa id Iraq soira minanangod o AS ontok toun 2003.


== Koubasanan ==
Kanada nopo nga haro mogisuusuai tinaru om koubasanan om kaanu nogi do kotinoligan parlambagaan montok dasar-dasar di poposuni kopgisuusuayon koubasanan mantad do mitos iso tinaru. Id Quebec, ogirot kopio jati diri koubasanan dau, hinonggo ogumu i moboros do boros Perancis popotolinahas do misuai kopio o koubasanan Qiuebec miampai do koubasanan Kanada Inggiris, nga i poinggonop nopo, Kanada nopo nga iso mozek koubasanan, ii nopo nga tinimungan do mogisuusuai sabkoubasanan kinoyonon, pribumi om etnik. Asaru posotolon do dasar-dasar Karajaan Kanada miagal do panamangan kolidasan tonggungon awam, cukai di akawas kopio montok popotikid kolumaagan, popoharam do hukuman mati, manamong do kopomogunaan kakamot tapui, popoluau do kopomogunaan dadah om kawawagu papasaga do pisasawaan di miagal jantina, ahal diti popokito do ponuduk sosial do misuai kopio evoulsi pulitik om koubasanan Kanada miampai Amerika Syarikat.

Solinaid do sajara, narahung o Kanada do koubasanan om kaadat-adato' British, Perancis, om pribumi. Ogumu hogot Koibutan Amerika, wonsoi om pomoinan pribumi pinosuang id pibarasan om maamangan sangaangadau id Kanada. Maya koubasanan, boros, inus, om muzik diolo, kakal po kirahung tagayo o tinaru pribumi kumaa jati diri Kanada.
Ogumu tulun Kanada mamagatang do kopogisuusuayan koubasanan om mongintong do pogun diolo sabaagi iso pogun di mogisusuai tinaru mantad po di timpuunon. Sundung ingkaa, koubasanan pogun diti arahung do koubasanan Amerika Syarikat tu minsomok do pogun dii om akawas nogi o ginumu koundolihon montok goduduo pogun. Majoriti i mundaliu momoboros Inggiris kumaa id Kanada mantad toun 1755 gisom 1815 nopo nga tulun Amerika mantad do Hopod om Tolu Jajahan; Maamaso om kalapas do Sangod Revolusi Amerika, minundaliu o 46,000 tulun Amerika di setia kumaa karajaan British kumaa id Kanada montok mampayat do 20,000 mogigion momoboros Inggiris i kogumuan nopo nga nokoundaliu no pogulu do Sangod Kagabasan. Mantad toun 1785 gisom 1812, ribu po kawagu tulun Amerika di minundaliu kumaa id Kanada montok do tana' di nabatos. Ontok toun 1812, kogumuan tulun Kanada i moboros Inggiris nopo nga minundaliu mantad bagas do hopod om tolu jajahan toi ko' tungkus diolo.
