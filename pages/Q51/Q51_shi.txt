Amnẓaw iqqurn utrim nna itussanen ula s iism n Antartika, iga yan umẓaw  izga ɣ wadday utrim n tkurt n wakal  . Izga umẓaw iqqurn ɣ tmnaḍt taquṭbit n wunẓul tlla iggin lquṭb ataram n trakalt, mk izga kullu ɣ tasutlt  taquṭbit taẓulant nna mysstl ugaraw n unẓul. tiwḍ tjumma  nns nnig n 14,000,000 kilumitr amkkuẓ mast  ittajan ad tg amnẓaw amqqran. Adfl yumẓ nig n 98% n Antarktika s wammas n turrut n km idal akk timnaḍin n umnẓaw ɣar adday n agafa n tamgzirt  taquṭbit n wunẓul.
Antarktika tga yan umnẓaw ɣ illa ugris n taɣart d waḍu ijhdn, nttat ad akk imqqurn ɣ imnẓawn zɣ tsga n unammas n uɣlluy. Antaktika llan dis tiniriwin n wakal zund tiniri taquṭbit, dars uṭṭun n waggaz n unẓar ɣ usggas nig n  2000 milimitr , ɣ uftas, maca ɣ tmnaḍin n ugʷns uṭṭun n waggaz n unẓar idrus  . tiskʷfal n tirɣi   ittuzmmamen ɣ antarktika tga nig n -89.2 n tskʷflt ɣ tmiḍi, maca anammas ɣ arbɛ anggaru (ɣ uzmz aṭṭas ismmiḍn  ɣ usggas) iga -63 taskʷflt ɣ tmiḍi. Izdaɣ ɣ antariktya nig 135 n ufgan,  ittzayad  uṭṭun ad jar  1000 d 5000 n ufgan ɣ usggas ɣ inzwayn n urzzu nna illan ɣ  ammas n umnẓaw . imaddarn  nna izdɣn ɣ tmnaḍt iga aṭṭas n wanawn n    ṭṭaḥalib d lvaktirya d lfiṭriyat d imɣan d kra n imudarn yaḍn zund ssus d tawkka  d izɛnafn n iḍarrn d lbaṭariq. Tsmma tmnaḍt lli ɣ llan imɣan  ɣ umnẓaw s ism n ttandra taquṭbit n iffus.
