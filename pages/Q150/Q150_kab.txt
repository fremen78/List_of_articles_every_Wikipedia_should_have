Tafrensist (s tefransist: La langue française) d yiwet n tutlayt taṛumantit, teṭṭef amkan wis sebɛa seg tutlayin i yettmeslayen ugar deg umaḍal akk s umkan wis 18 deg tutlayin i yettmeslayen ugar am tutlayt tayemmat,  ttmeslayen-tt nnig n 80 n yimelyunen n midden deg umaḍal akk am tutlayt tamezwarut, d azal n 190 n yimelyunen n midden am tutlayt tis snat, d azal n 274 n yimelyunen deg umaḍal akk.  Imsawalen-a llan di 54 n tmura deg umaḍal. 
Tafṛansist, am tutlayin nniḍen n umaḍal am tapurtugit, d [[taspenyult], taṭelyanit, tkatalantit d tṛumanit, tekka-d seg tlatinit,d tutlayt n Tmurt n iṚumaniyen. Ttemcabin-tt daɣen deg wazal-a kra n tutlayin tiɣermiyin am tutlayt taqṭanit neɣ taksitantit, d tutlayt i ttmeslayen yimezdaɣ n Usṭanya deg wenẓul n Fransa, d tutlayt n Neapolitan d tutlayt n yimezdaɣ n Naples deg Ṭṭelyan, d tiyaḍ.
