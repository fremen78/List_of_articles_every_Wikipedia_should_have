Australia nopo nga olohou do rasmi sabaagi Komanwel Australia, iso pogun id Hemisfera Selatan ii mantad do tana tagayo om iso bonua di tokoro id pomogunan, miampai kiwaa pulau id Lautan Selatan, Hindi om Pasifik. Pogun-pogun sombol nopo nga miagal ko Indonesia, Timor Leste om Papua New Guinea id utara, Kepulauan Solomon, Vanuatu om New Caledonia id timur laut, miampai New Zealand if tenggara.

Tana tagayo Australia nopo nga noiyon do tulun sandad Australia mantad 60.000 toun po. Soira do kinotombuluyan soroinsan-insan do mamagapon mantad koibutan om papadagang mantad Eropah, nga tumimpuun do abad ke-17, sampaping do kosilahon tana tagayo diti nopo nga nongoi do koobian British ontok toun 1770, om New South Wales nopo nga pinosiliu sabaagi iso tana pinorinta montok ukum mibagal poimbida' ontok 26 Milatok 1788. Soira nokogumu mogigion om kinoyonon-kinoiyonon wagu noi porinta'o haro kaagu limo kinoiyonon ii kagal-agal do Tanah porinta'on ponguasaan sondii noturidong mogisusuhut solinaid abad ke-19. 


== Tadon ngaran ==
Ngaran 'Australia' nopo nga mantad hogot boros Latin, australis i kikomoyon do id selatan. Ka nopo lagenda nga kokomoi do "iso tana ii au notutunan id selatan" (terra australis incognita) ii sinusui do tulun mantad zaman Rom po. Bonua diti nogi nga kiwaa id suang geografi zaman pintangaan, nga okon ko miimpohon do toilaan kopiipio. Bontok boros ula Belanda, Australische, nopo nga ginuno mantad di 1638 numaan di pegawai-pegawai Belanda id Batavia i monukku kumaa do tana id selatan i wagu do nokito. Pomogunoon koiso hogot "Australia" id boros Inggeris nopo nga iso pomodolinan di 1693 montok Les Aventures de Jacques Sadeur dans la DÃ©couverte et le Voyage de la Terre Australe, iso novel Perancis ii sinuat ontok toun 1692 mantad di Gabriel de Foigny miampai ngaran pena do Jacques Sadeur.  


== Sajara ==


== Geografi ==
Dana Australia nopo nga kilinaab 7,,696,850 kilometer pasagi ii poingiliu id Plat Indo-Australia. Nosorilian iti do Lautan Hindi, Selatan om Pasifik om opitongkiad mantad Asia om Rahat Arafura om nogi Rahat Timor. 


== Ekonomi ==
Ekonomi id Australia nopo nga naralatan do gaya barat ii ongopuhawang miampai KDNK per kapita ii lobi akawas tokuri mantad do United Kingdom, Jerman om Perancis id sogi kuasa beli. Pogun diti nopo nga id taang kotolu id suang Indeks Pombangunan Tongoulun 2005 mantad di Pertubuhan Bangsa-Bangsa Bersatu, om koonom id indeks kualiti koposion sompomogunan 2005 mantad di The Economist. 


== Demografi ==
Kogumuan mogiigiyon Australia nopo nga kiikiro soginumu 20.6 juta tulun i kiraha do peneroka-peneroka abad ko-19 om ko-20. Kogumuan nopo diolo nga mantad do Great Britain om Ireland. Ginumu mogiigiyon Australia nopo nga nokoinggumu do ingapat mantad di tohuri Perang Pomogunan I, om nounsub do palan imigrasi.
