Arxmides neɣ Arcimides (s tagrikit taqburt : Ἀρχιμήδης, s talatint : Archimedes), illul deg ahat 287 Uqbel Ɛisa deg Sičilya, yemmut deg 212/211 Uqbel Ɛisa deg Sičilya daɣen, d amusnak ed amesnulfu i mucaɛen maḍi deg Legrik tagelsant. 


== Tudert ==
Arxmides yezdeɣ tuget n tudert-is deg Syrakusa (d tamdint tagrikit meqqren akk deg Sičilya), anda yella d imqerreb n ugellid-is Hieron II. Yella icɣel-d d inadiyen ussnan deg waṭas n taɣulin em tanzeggit, tasengama ed tmikanikt. Yella yetturar tamlilt d taxatart deg aḥuddu n Syrakusa mgal yirumyen deg 213 Üqbel Ɛisa (yebna tinaεurin n ṭṭrad), deffir irumyen kecmen ɣar Syrakusa deg 212 neɣ 211 Uqbel Ɛisa, irumyen i t-yenɣan Arxmides deg tamdint-is. 


== Ayen yexdem ==
Arxmides yufa-d wassaɣ ger tajumma ed ubleɣ n takurt ed awlellu-is, yufa-d daɣen umenzay n Arxmides ed tafetlit n Arxmides.

=== Tafetlit n Arxmides ===

Arxmides i d-yesnulfan tafetlit-is (tafetlit n Arxmides); d tnaεurt tella i wakken ad tessnekreḍ aman, d yiwent seg talɣiwin n ujeɛbub deg-s talɣa n tafetlit yekna akked tiɣmeṛt n ahat 45 n tfesniwin (45°) yebbeɣ deg waman ɣer yixef-is amedda. Amussu n tnaεurt-nni yessuli aman deg ujeɛbub.
