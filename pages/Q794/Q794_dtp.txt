Iran toi ko' ngaran di poimbantug nopo nga Republik Islam Iran (Parsi: جمهوری اسلامی ايران, Jomhuriye Eslamiye Iran), oilaan nogi sabaagi Parsi id pogun-pogun kotonobon, iso pogun id Asia Kotonobon. Pogun diti nopo nga miwolit do Armenia, Azerbaijian om Turkmenistan id koibutan, Pakistan om Afghanistan id kosilahon , om Turki om Iraq (wolit Kurdistan) id kotonobon. Iran nogi nga miilang do Teluk Parsi miampai Kuwait, Iraq, Arab Saudi, Bahrain, Oman, Qatar om nogi Emiriah Arab Bersatu. Kotumbayan poimbantug nopo nga Islam Syiah om boros poimbantug nopo nga Boros Parsi.
