Lwis Pastur (s tfransist: Louis Pasteur; ilul nn ɣ 1822, immut nn ɣ 1895) iga yan umassan Afransawi iɣran tacimit ilan bahra atig ɣ umzruy, iskr yat tagrawla ɣ igr n twssna, tiwuriwin nns jjnjmnt gigan d mddn slawan lliɣ d yufa agzi n yiṣiḍ, ɣ wattayn n 1860 iga amazwaru lli d yufan is nẓḍaṛ ad nenqqa   irctuyn (germes/جراثيم) s tirɣi, ɣaylli d ifkan tlalit i yaṭ tmhlt lli s ttinin "Tasppasturt" (Pasteurization), yaf daɣ tazdayt da illan ngr irctuyn d tmuḍan, d wahli matta kullu ɣaylli yufa. 


== Tinqqaṭṭ ɣ tudrt nns ==
Ɣid kra n iẓmmuẓẓal mqqurnin ɣ tudrt n umassan ad: 
1822 : ilul nn Lwis Pastur ɣ Dal (Jura).
1839 : Assnti nns ɣ ukulij agldan n Bzansun.￼
1842 : Yumẓ lbaclur nns ɣ tusnakt.
1843 : izri s ENS (école normale supérieure).
1845 : Yumẓ alisuns ɣ twssna. 
1846 : imurs ad ig aslmad n ufizik ɣ ulisi n Turnun macc iɣama ɣ ENS.
1847: Yumẓ ḍḍukṭuṛa ɣ twssna 
1848 : iga aslmad n ufizik ɣ ulisi n Diju fadd s uslmad amssiws ɣ tcimit ɣ tsddawit n Stasburg. 
1849: Itahl d lalla "Marie Laurent". 
1850: tlalit n illis Jyan (tmmut sul ɣ 1859) 
1851: Tlalit n yiwis Jun baptist 
1852: kra n isnubbucn imaynutn ɣ tzdayt iẓḍṛn ad ittilin gr talɣiwin ikrisṭaliyn. 
1853: tlalit n illis Sisil, ittsma amnay n "Ordre impérial de la légion d'honneur". 
1854: iga addwayyan n tsddawit n twssna n Lill 
1855: assnti n tiɣriwin f Tsmtunt. 
1856: assnti n isnubbucn ɣ Tsmtunt talkuḥulit (fermentation alcoolique)  
1857: iga anmhal n ENS n lill nit￼ 
1859: Tasmɣurt n "phisiologie experimentale" ɣ tkadimit n twssna f twuriwin nns ɣ igr n tsmtunt  
1860: Tasmɣurt n "Jecker"  
1862: Tasmɣurt n "Alhumbert", yufa is d imikrubn lli llanin ɣ tɣawsiwin pasturnin kkant d ad iɣʷẓan aḍu.  
1863: Naplyun wis kṛaḍ ittr t ad as iɣr f tmuḍan n unẓiẓ. 
1865: Tasppasturit.  
1866: ifsr kra n twriqin f unẓiẓ.  
1877: awal f ibẓḍan 
1878: isawl f irctuyn d tazdayt nnsn d tsnijjit , tasuksidt ar tssidrus ɣ thrci n kra n umikrub, isawl daɣ f uzray n imikrubn s idammn d twla n wactal (iḥcmiyyn). 
1880: awal f ṭṭaɛun. 
1885: Agzi amzwaru mgal iṣiḍ dar ufgan. 
1888: Aṛẓẓum n usinag n Pastur. 
1895: immut ɣ "Villeneuve" ɣ 28 n cutambir.
