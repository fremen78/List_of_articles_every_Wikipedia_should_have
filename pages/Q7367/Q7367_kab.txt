Tadast (Assaɣ ussnan: Culicidae) d tawacult n yibureɣsen yeṭṭafaten tafesna n  tasenfriwin, Dɣa d tiwetmin n tafast iy tsummun idammen n wemdan akken i tga d yiwen seg ibureɣsen inesnasen (itsummun idammen) yellan s tuget yakk deg umadal. Anecta yettaǧǧa-tt ad tessawaṭ aṭas n telmas n waṭṭanen am umalarya.
Awtem n tadast d win i d-itekksen učči-ines seg wemɣus yakk d uselɣaɣ n yimɣan ma d tawtemt tettsummu idammen acku gan d imsefkiyen i tuwwit n tmellalin-is 


== Asesmel ==
Llant ugar n 3000 n telmas n tadasin yeṭṭafaren azal n 35 n tewsitin. Dɣa tawacult n tadast bṭant ɣer snat n wadu-twaculin:

Tinufilawin
Bironella
Chagasia
Tidasawin
Abiba
