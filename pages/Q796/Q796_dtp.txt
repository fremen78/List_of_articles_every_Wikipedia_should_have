Iran (Boros Arab العراق al-‘Irāq atau al-Erāq,), Boros Kurdish عيَراق), ii poimbantug nopo nga Republik Iraq, iso pogun kosilahon tanga id Asia Barat-Daya ii kahangkum do kogumuan Mesopotamia, om kahangkum do kotonobon rahat konuluan Zagros om boogian kosilahon mantad Padang Pasir Syria. Ii miilang do wolit miampai Kuwait om Arab Saudia gisom id kabaatan, Jordan id kotonobon, Syria id kotonobon rahat, Turki id koibutan om Iran (Kurdistan Wilayah)(Parsi) id kosilahon. Haro boogian ii okikip id disan rahat id Umm Qasr id Teluk Parsi.
