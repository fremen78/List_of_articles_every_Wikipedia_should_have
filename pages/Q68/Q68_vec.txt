El computer computerador o calcołador 
el xe un aparecio enformàdego chel ne permete tratar łe enformassion ndando drio serte proxedure prezixe. El xe un sistema digitałe co tecnołoja microełetronica chel xe bon de prozessar dati a partir de un grupo de istrussion ciamà programa.
Na strutura base de un computer ła ga rento: microprocessor (CPU), memoria, dispositií I/O (input/output, che voria dir rento/fora).
In sostansa el computer el xe na màchina che ła xonta el software co el hardware. El software l'è quel che se monta dentro su par l'hard disk, el hardware l'è quel che ti te pol veder e tocar.
Tra i software più famosi potemo trovare: el pacheto Microsoft Office, el sistema Microsoft Windows, el navigator Mozilla Firefox e i driver dele aparechiature colegade alla machina.
