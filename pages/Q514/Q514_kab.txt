Asemmili d yiwen n ufurk n tesnudert izerrew (yeɣɣaṛ) talɣa d tmuski n imuddiren yakk d yimuren-nsen (igmanen d yizeṭwan isnudranen). Asemmili s timmad-is yeqqen ɣer tesnamgunt (Embryology) d usemmili imserwes (Comparative anatomy) yakk d tesnudert tanamhazt, Sɣur timhalin i d-tzerrew tussna ayyi, yettnulfaw-d usemmili deg wakud meẓẓiyen s waṭas (tasnamgunt) yakk deg wakud ɣezzifen s waṭas (tamhazt), dɣa asemmili n ufgan iga d yiwen gar tisnajjyanin tigejdanin
Asemmili yebṭa ar sin imuren igejdanen gan d Asemmili amatay d umikruskupay 
Asemmili amatay izerrew akayed n imuren d yigmamen n uɣersiw s usekkud anda yella deg-s afurk n usemmili ajumman. Ma deg usemmili mikṛu skupay ttusxedmen deg-s ifecka isekdanen deg tezrawt n izeṭṭiwen n tmuskiwin yemgaraden akken daɣen i yzerrew tikwatin  
Asemmili yettusken s tigzi tafarayt i twuriwin n yigmanen d tmuskiwin n tfekka talsant. Tirrayin n tezrawt n usemmili d tid yettusekren (yettusgerzen) s waṭas seg ukayad n iɣersiwen s usemmili n tfekkiwin-nsen d imeɣsuyen-nsen alammi tewweṭ ar usemres (asexdem) n titekniyin n tssugent tisnajjyanin g tsut tis 20 gar-asent azenzar X d tenzayt d tssugent s uzenzen adekran. Asemmili d tsenwurit ssulɣen-d tayuga n tussniwin yettwazrawen yakked (lwaḥa) deg waṭas n tmuzzag. Anda snat n tussniwin-a zerrweny tamuski n tfekka d twuriwin-is s umdafar

== Ifurken n usemmili ==
Asemmili ajumman d afukr ilan (yesɛan) azal meqqren i tezrawt n yisdagen n usemmili yettwaẓren s tefrer seg izerrigen uffiɣen n tfekka. Ayen yettaǧǧan imessujiyen d imeftaken ad ssazeln timersiwt d tegnit w ad semmilin timuskiwin tilqayanin yeqqnen ar yisdagen-a, Dɣa irem n ujumman d irem anilan yemmal-d d akken timuskiwin zgant-d zdat n tiggi d usfel n tfekka.
Ma d asemmili amikṛuskupan yeqqen ar userwes n tmuskiwin n usemmili (timikruskupyim) deg imuddiren yemgaraden.
