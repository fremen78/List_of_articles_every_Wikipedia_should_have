Sepanyol, ointutunan sabaagi Kingdom of Spain, nopo nga pogun id kabaatan kotonobon Eropah miampai wilayah id Afrika Koibutan. Ii noo nga pogun bobos agayo Eropah Kaaaatan om pogun Koisaan Eropah kaapat i kiaa ogumu mogigion. Papahangkum soboogian tagayo Semenanjung Iberia, wilayah nogi dii nga kohompit no Kepulauan Canary id Lautan Atlantik, Kepulauan Balearic id Rahat Mediterranean, om kakadayan autonomiCeuta om Melilla id Afrika. Sepanyol Semenanjung nopo nga miwolit d koibutan miampai Perancis, Andorra, om Teluk Biscay; id kosilahon om kabaatan miampai Rahat Mediterranean om Gibraltar; om id kotonobon miampa Portugal om Lautan Atlantik. Ibu negara om kakadayan bobos agayo do Sepanyol nopo nga Madrid, om koiyonon kakadayan suai kohompit no Barcelona, Valencia, om Zaragoza.
Ontok zaman purba nokopogulu, Semenanjung Iberia nopo nga ionon do tulun Celt, Iberia, om tinimungan momogun pra-Romawi di suai. Miampai panaladan Rom id Semenanjung Iberia, noturidong o wilayah Hispania. Kalapas proses Romanisasi om Kristianiti Hispania, kinorotuon Empayar Rom Kotonobon minogowit kumaa kinosuangon puak mantad Eropah Tanga, kohompit no Visigoths, i minombontuk Porinta Visigothic poinlongkod id Toledo. Id kotimpuunon abad kawalu, nasangod do khalifah Umayyah o soboogian tagayo semenanjung, om maso pomorintaan Siam pogulu, nosiliu kuasa dominan o Al-Andalus i poinlongkod id Córdoba. Piipiro porinta Kristian nokoimbulai id Kosilahon Iberia, miagal do Asturias, León, Castile, Aragon, Navarre, om Portugal; om minaparalad yolo ketenteraan miampai berkala kumaa id kabaatan om nogi koiyonon kawagu i ointutunan sabaagi Reconquista, i gumaga kopomorintaan Silam id Iberia, om nokodimpot dimpak miampai panaladan Porinta Nasid Granada ontok 1492. Kopisaan dinasti Mahkota Castile om Mahhkota Aragon ontok toun 1479 id siriba Raja Katolik nopo kaka nga sabaagi kopisaan de facto Sepanyol sabaagi pogun bansa.


== Pomungaranan ==
Ngaran "Sepanyol" nopo nga naanu mantad do Boros Portugis, Espanhol ii iso bontuk boros adjektif om demonim do ngaran koiyonon Espanha. Ngaran "Espanha" nopo dii nga pinasandad do ngaran Latin Hispania ii nopo nga morujuk do koiyonon ii natadbir do Republik Rom poimpuli do Semenanjung Iberia ontok di pogulu po.
