Ivirist (ittyuga as yism: Sagarmata dar ayt Unippal s unamk n "Afa n ignna", d: Kumulungma dar ayt ttibit s unamk n "Lall n ddunit") iga yan udrar illan ɣ iwtta ngr Unippal d ttibit, iga agzzum sɣ isddi n udrar n Yimalaya lliɣ llan iskawn n yidrarn akkʷ attuynin ɣ ddunit, nttni waḥdu tn ka ad issurfn 8000 n umitru ɣ tattuyt, lkmn akkʷ ar 8848 n umitru. 

Imzwura da ẓḍarnin ad nrun adrar ad gan tn d Idmund Hillariy ssɣ Nyuzilanḍa, d Tinzing Nurgay ssɣ Unippal nit, d mayad ɣ usggʷas n 1953, mqqar ɣik ann hati mnck d yan ad ifran s tudrt nns lliɣ ran ad dis ɣlin.


== Aẓuṛ n Yism ==
Ism n udrar ar d ittiwrri s yan unmhal Angliziy n tinufa tirakalin n Bngal ɣ lhnd ism as "Sir Jurj Ivirist", acku ur ufin ingliziyyn ḥtta kra n ism yaḍni ittyawssan sis udrar ad, fkn as ilmma ɣwad. Waxxa Ccinwa ar sɣuyyun f mayad, acku illa yad yan ism win tmazirt igan aqbur tili tugga f tilit nns ɣ yat tkaṛḍa tacinwit 300 n usggʷas ayad. ism ad iga "kumulangma".    


== Laẓun n Tmttant ==
Laẓun n tmttant (Death zone) tlla ɣilli ɣ tssurf tattuyt 8000 n umitru acku f uṣmmiḍ iqḍɛan da ɣinn ittilin, sul iẓḍaṛ kra n yan ad iccḍ iḍṛ, acku f uɣris da iddaln nit akal, bla ad nttu asadd n tḍaṛut (Atmospheric pressure) da issidrusn anck n uksijin ɣ waḍu.  
Yan imikk uggar n 200 yan ad immutn, ittyafa is d tiggti nnsn tmmut ɣ uflla n 8000 n umitru. 


== Aɣaras n wawwan ==
Wanna iran ad yawn i udrar llan dars sin iɣarasn, swa ad ikk Nippal (win yiffus) niɣ ad ikk Ttibit (win ugafa), aɣaras n yiffus ad irxan d nttan ad bahra ttkkan mddn.
