Api iolah prosses oksidasi capek nan tajadi ka suatu material dalam pambakaran kimiawi, nan manghasilkan paneh, cahayo, jo hasil reaksi kimi lainnyo. Proses ko nan labiah lambek bisa dicaliak di proses pangkaratan basi. Api barupo energi baintensitas nan bavariasi dan mamiliki bantuak cahayo (panjang galombangnyo di lua spektrum visual sahinggo dapek diliek dek mato manusia) dan angeknyo dapek manimbukan asok.

Warna api dipangaruhi dek intensitas cahayonyo nan biaso digunoan untuak manantua apokah suatu baha bakar tamasuak dalam tingkatan kombusi sahinggo dapek digunoan untuak kaparluan manusia (misal digunoan untuak bahan bakar api unggun, parapian atau kompor gas) atau tingkek pambaka nan kareh nan basifaik sangek pahancua, mambaka jo caro indak takontrol dapek marugikan lingkuanggan sakitar (misal pambakaran pado gedung, hutan dan lainnyo).

Panamuan caro mambuek api marupokan salah satu hal nan paliang baguno dek manusia, karano jo api, golonggan hominids (manusia dan kerabaiknyo contohnhyo kera) dapek aman dari hewan buas, mamasak makanan, dan mandapek sumber cahayo sarato manjago dirinyo tetap angek. Bahkan masih banyak masyarakat zaman kini ko nan primitif maanggap api sabagai sumber kahidupan sagalo makhluk iduik. Api tabantuak katiko udaro tarutamo oksigen mangalami panggabuangan jo benda nan mudah tabaka atau bahan baka nan kamudian mandapek energi paneh nan kamudian mamicu reaksi kimia antaro oksigen, bahan bak, jo energi paneh. Api nan ditamukan di bumi babeda jo nan basarobok di bintang, api di bumi umumnyo manghasilkan energi nan ketek dibandianggan energi api dari bintang.
