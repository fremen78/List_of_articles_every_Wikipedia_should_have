Api (Jawi: ‏اڤي‎code: ms is deprecated ‎) ialah proses oksidasi pantas dengan menghasilkan cahaya, haba, asap dan pelbagai hasil tindak balas kimia seperti karbon dioksida dan air. Api lazimnya diklasifikasikan sebagai pembakaran minyak atau bahan api dalam keadaan terbakar (misalnya unggun, pendiangan atau dapur) atau satu pembakaran ganas, pemusnah dan tidak terkawal (seperti dalam bangunan atau api liar). Penemuan bagaimana membuat api dianggap salah satu kemajuan peradaban manusia yang penting membenarkan hominid yang lebih tinggi perkembangannya mengusir binatang liar, memasak makanan dan mengawal sumber cahaya serta memanaskan badan sendiri.
Perkataan "api" diturunkan daripada akar Proto-Austronesia *Sapuy.
