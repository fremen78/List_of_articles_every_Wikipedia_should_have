Api tu reaksyen kimia ti mansutka penampak enggau pengangat. Api nyadi lebuh utai jampat beoksida, tauka lenyau elektron, lalu ngelepaska mayuh tenaga. Api kadang-kadang bisi guna, tang mega balat bendar mai penusah laban iya tau ngasuh rumah, pun kayu enggau utai bukai angus nyadi abu. Api tu ulih ngerusak kandang endur ti besai dalam kandang beberapa minit aja. Tiap taun orang bisi mati laban penusah ketegal angus. 
Api ulih dipunka enggau chara ngasuh sesebengkah utai angat ngena mayuh macham chara. Pengudah utai nya dipanas ngelui ari tikas pengangat ti dikumbai poin penunu, iya deka berengkah angus, lalu ngepunka api. Sekeda chara ti suah dikena ngepunka api nyengkaum ngusap kayu enggau jampat amat, ngaga percikan api ngena chara nimbak batu api ngena besi, tauka ngena mancis tauka pemanchar api. Matahari enda ngaga api. Tang, iya ngeredakka atom hidrogen ngambika ulih ngelepaska tenaga nengah proses ke dikumbai fusyen nuklear.


== Guna ==
Api tau bendar beguna enti iya dijaga enggau manah. Ari suba beguna amat orang ulih ngaga api. Orang begunaka pengangat iya ngambika ulih angat maya hari ti chelap. Iya mega dikena manduk dagin. Penampak iya nulung orang meda ba endur ti petang sereta ngenakutka jelu ti makai sida.


== Penusah ==
Enti kulit mensia negu api, kulit nya tau angus, lalu tu tau begunaka awak ngambika gerai sereta muai bekau nya. Enti api ngeluarka mayuh asap, mulut orang patut ditutup ngena kain ti basah, laban orang tau luput lebuh enseputka asap ti kelalu mayuh. Indah api mit, baka ari insap, patut dijauhka ari sebarang minyak ti mudah amat nangkap sereta ngerembaika api, baka minyak petrol, tauka ngasuh meletup, baka bubuk.
Enti api enda dijaga enggau manah, iya tau bendar mai penusah. Siti api kampung kekadang nganguska beribu batu persegi tauka kilometer. Kampung tau angus enti api enda ditagang. Ninting taun, kandang endur kampung ti besai rusak ketegal angus. Tu selalu nyadi maya musim panas.
