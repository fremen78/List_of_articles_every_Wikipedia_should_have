Buul sipak nopo nga sipoot tinimungan i pomoinan do duo tinimungan hinonggo tikid tinimungan nopo nga haro 11 mamamain. Momoguno i mamamain do gakod diolo montok manamong buul id posorili do gana i kibontuk do apat pasagi tapat, id sakot toi ko' sakot di winonsoi-wonsoi.
Koubasan nopo nga i mananamong do gol no milo mongigit buul momoguno do longon id suang kinoyonon gol dau i roitan do kutak penalti. 10 mamamain di suai nopo nga milo momoguno oinsanan boogian tinan diolo suai ko' longon, miagal do gakod i gunoon montok manandang, kangkab montok manamong, om tulu gunoon montok monungu do buul. Kalantoi o tinimungan di kaanu gol di bobos togumu id kolimpupuson do pialaan. Nung naawi timpu om goduduo tinimungan dii haro gol di miagal ginumu nga milo wonsoyon o pomilian, inwoguan o timpu toi ko' tandangan penalti, tumanud do format panganjul pialaan dii.
Inwoguan do Persekutuan Piisaan Buul Sipak Sompomogunan do kooturan pialaan do monikid toun.


== Sajara' ==

Mogisuusuai kawo pomoinan buul pinointutun ontok Timpu Pintangaan Eropah. Ontok abad ko-19, nointutunan o pomoinan buul moden id sikul-sikul United Kingdom (i kiharo karalano' pomoinan di mogisuusuai tumanud sikul). Mantad dii, karalano pomoinan dau nga lobi oulud. Pinointutun o kooturan-kooturan timpuunon moi do aagal karalano' pomoinan di mogisuusuai om popouhan di mamamain. Pinointutun do British o sipoot buul sipak id Nusantara maya Risalat Kooturan Buul Sipak i pinungaranan do Inggeris Football om pinosoliwan id Singapura ontok toun 1859.
Pinoturidong do piipiro kalab buul sipak o Piisaan Buul Sipak England ontok 1863. Nakalantoi o piisaan diti do monupu kooturan-kooturan formal buul sipak i kumoiso nogi id pomogunan. Nokoburu kawagu o kooturan-kooturan diti suminiliu kooturan-kooturan buul sipak tadau do baino.
Uludon do Lambaga Piisaan Buul Sipak Sompomogunan o nunu nopo kokomoi Kooturan-kooturan Buul Sipak. Pinoturidong o Lambaga diti ontok 1886 om haro walu o surupu - apat mongongobi do FIFA, apat mongongobi piisaan buul sipak England, Scotland, Wales om Panansawat Ireland. Nunu nopo kosimbanan do Kooturan-kooturan Buul Sipak nga nuru do haro kasagaan mantad Lambaga maya pomilian di bobos ogumu.
Tumanud ponoriukan di winonsoi do Fédération Internationale de Football Association (FIFA), iso kotinanan momuruan buul sipak, i pinosoliwan ontok musim bunga 2001, lobi do 240 riong tulun i mamain buul sipak i popohompit do lobi 200 pogun id pointongkop pomogunan. Kooturan dao i ouhan om momoguno kakamot di okuri minonguhup do kinarayadan om kinointutunan do sipoot diti. Pogulu do abad ko-21, lobi do 250 riong tulun di mamain do sipoot diti id lobi do 200 pogun.


== Karalano' pomoinan ==
Buul sipak nopo nga iso pomoinan i popohompit do duo tinimungan i kiharo do 11 tulun mamamain om tikid tinimungan mingoos do lumantoi id pialaan dii miampai maganu gol di lobi ko' mantad gol do puru di monongkiala'. Iso gol nopo nga asil do buul di pinaatod i katalib do garisan gol i poingiliu id pialatan do torigi gol om palang gol. Buul sipak nopo nga pomoinan momoguno do goduduo gakod, nga pasaga'on o mamamain do momoguno oinsanan boogian tinan diolo suai ko' longon. Mananamong gol nopo nga pasaga'on do momoguno longon montok mamagakom do buul id suang kinoyonon penalti.
Tinimungan di kaanu lobi ogumu gol nga kalantoi id pialaan dii. Nung goduduo tinimungan kaanu gol di miagal, pialaan di nopo nga seri. Id suang pialaan naala' apatai, montok manantu' isai i kalantoi, ruhangon o timpu 30 minit nung goduduo tinimungan haro gol di miagal id suang timpu 90 minit mantad. Tinimungan di kaanu gol di poinlobi id suang timpu diti nopo nga kalantoi id pialaan dii.
Nung amu kasari natantu' isai i nakalantoi sundung do noruhangan no timpu, powonsoyon o tandangan penalti. Tikid tinimungan tahakon do 5 tandangan penalti. Tinimungan di kaanu penalti di poinlobi nopo nga kalantoi id pialaan dii.


== Pogialaan sompomogunan ==
Piboi'an sompomogunan di bobos tagayo id suang gana buul sipak nopo nga  Piala Pomogunan FIFA i poonjulon do FIFA. Miagal di noilaan no, pohoroon o Piala Pomogunan FIFA tikid do apat toun om nokotimpuun id Uruguay ontok toun 1930. Lobi do 200 pogun mogiala'-ala' id taang panantaban tumanud zon pogulu do kosuang id taang tohuri. Brazil nopo nga tinimungan di bobos nakalantoi id suang Piala Pomogunan FIFA tu nokosiliu bontugan do inlimo, kinalantayon di kawawagu nopo nga ontok toun 2002.
Piboi'an sompomogunan di otuo kopio id suang gana buul sipak nopo nga Copa América i pinaanjul do CONCACAF. Miagal di noilaan, pohoroon o Copa América tikid do duo toun om nokotimpuun id Argentina ontok 1916.
