Tga Ddzayr (s tɛrabt : الجزائر) yat tmazirt illan ɣ Tfriqt n ugafa, ism nns amaddud iga t Tagduda Tamgdayt Taɣrfant Tadzariyt (s Tɛṛabt: الجمهورية الجزائرية الديمقراطية الشعبية). Tamaẓunt nns tga tt Ddzayr .
Ism nns ikka d taguri taɛrabt "الجزائر", nna mi iga unamk tigzirin. Ism ad iwint t id tgzirin llanin f uftas n tmazirt n dzayr.


== Tarakalt ==
Tamazirt n Dzayr tga tamazirt nna akk imqqurn ɣ timizar tigrakalin s tajumma nns, tga tamazirt tis-snat imqqurn ɣ tmizar n umnẓaw n Tafriqt, teṭṭaf tamazirt n Dzayr ɣ tsga n iffus ayda imqqurn n tnzruft ism nns Sehra.


=== Tarakalt tadeblant ===
Tbḍa tamazirt n Dzayr f 58 n twilayin, kraygat tawilayt teṭṭaf agraw aɣrfan f ixf nns lwali. Kraygat tawilayt tbḍa f asunen, kraygat asun ibḍa f tɣiwanin. Tikklt tamggarut ɣ tettwabḍa gis tamazirt n Dzayr tga ɣ useggas n 2019.
