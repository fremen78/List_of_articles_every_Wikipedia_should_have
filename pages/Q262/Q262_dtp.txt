Algeria toi ko' ngaran rasmi Republik Demokratik Rakyat Algeria  nopo nga iso pogun id Afrika Utara poingwiliu id koibutan kotonobon benua Afrika miampai pantai id Laut Mediterranean id koibutan, miwolit miampai Tunisia id koibutan kosilaon, Libya id kosilahon, Niger id kabaatan kosilaon, Mali om Mauritania id koibutan kotonobon, om Maghribi id kotonobon.
Algeria nopo nga pogun koduo tagayo id benua Afrika.


== Prasarana ==


=== Pondidikan ===
Pondidikan id Pogun Algeria nopo nga nuru montok ngai mogigion di kiumul 6 gisom 15 toun.  sundung pia di nga opintanga no tangaanak sikul pogun diti it kaanu sumikul id taang menengah. Gisom toun 2015, pogun Algeria kiharo 92 institut pasca menengah miampai 48 university.
