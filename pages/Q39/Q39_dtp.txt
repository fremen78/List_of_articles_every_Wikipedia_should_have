Switzerland ([swit.zĕr.land]; Jerman: die Schweiz, Perancis: Suisse, Itali: Svizzera, Romansh: Svizra), nopo nga poinhimag id piisaan Switzerland (Confœderatio Helvetica id boros Latin, om kod pogun id ISO nopo nga CH), nopo nga iso pogun id Eropah id kiwaa aiso pantai om kiwaa kalaaban do 41,285 km om kiwaa soginumu 7.5 riong tulun mogigion. 
Kotoinaan Switzerland nopo nga baagi kumaa 26 kanton, onom mantad di nopo nga kasasari kosiliu sabaagi ''opintanga kanton" tu kiwaa kopitongkiadan do tolu kanton om bagas dau nopo nga miilang iso no wilayah id suang Majlis Negeri-Negeri. Ibu negara nopo do Switzerland nga iri watas do Bern om watas tagayo nopo dau nga Zurich (id nakaanu kinoyonon di haro kualiti koposion di osonong mantad toun 2006 gisom 2007).
Switzerland nopo nga miwolit miampai Jerman, Perancis, Itali, Austria om karajaan tokoro Liechtenstein. Koulunan Switzerland nopo nga mogikaakawo boros om kiwaa apat boros rasmi ii nopo nga boros Jerman, Perancis, Itali om boros Romansh ii au kopio nointutunan.
Opuhawang nogi Switzerland miampai do sajara sabaagi pogun ii neutral miampai au mongiro do timpu miparang toi ko damai (au insan kohompit id perang kumaa kuasa asing mantad toun 1815). Mantad dii, nosiliu o Switzerland sabaagi do tuan walai kumaa do mogikaakawo pertubuhan sompomogunan miagal ko Pertubuhan Bangsa-Bangsa Bersatu, sundung do ibu pejabat nopo nga id Bandaraya New York, nga kiwaa nogi jabatan id Switzerland. Taandakon o Tadau Merdeka ontok 1 Magus 1291 id Switzerland; tumanud sajara pogun diti, di timpuunon nopo nga iso piomungan, nga tumilombus nosiliu do persekutuan mantad toun 1848. 1 Magus nopo nga sabaagi koundarangan umum hinombo bank om pejabat pos om nogi pontadbiran awam nopo nga pointobon.  


== Pomungaranan ==
Schweizerische Eidgenossenschaft nopo nga istila boros Jerman ii haro id dokumen rasmi. Ngaran boros nopo id Latin nga "Confœderatio Helvetica" (piomungan Helvetik) om kod pogun nopo nga "ch" montok internet om sosokot kulita nopo nga kokomoi Helvetii, ii poingolong kaum Celt kuno ii nokoumbal do mogiigiyon id banjaran Alp; ngaran diti nopo nga poinsuat id Kitab Vortaro toun 1923.


== Geografi ==
Nokoinlaab id utara om sisi selatan konuluon Alp id Eropah barat tanga, Switzerland ii kolimput do kopogisuusuaian landskap om iklim id koiyonon terbatas 41,285 kilometer persegi. Populasi nopo nga kiikiro 8 juta miampai purata kepadatan mogiigiyon kiikiro 195 tulun per kilometer persegi. Lobi kumaa do boogian selatan konuluon pogun dii, mingkuuri o mogiigiyon id boogian utara. Kanton nopo nga ii taagayo kopio id Graubunden, i poimponu do poingiliu id Konuluon Alps miampai ginumu tulun nopo nga 27 kilometer persegi.

=== Eksekutif ===
Sistom pomorintaan Switzerland nopo nga unik. Nosiliu iti sabaagi do pogun persekutuan mantad di 1848. Momoguno do sistom demokrasi potilombus om pomorintaan dau sondiri miampai turu tulun puru ii nopili do Porhimpunan Persekutuan. Koturu-turu tulun dii nopo nga momuruan do jabatan gulu. Status diolo nopo nga arait do montiri. Ii kagayat ginawo nopo nga, turu tulun di nopili nopo nga mogisoosoli do sumiliu presiden. Jawatan sabaagi presiden nopo nga solinaid do sontoun.

== Ekonomi ==
Switzerland nopo nga kiwaa ekonomi di stabil, makmur om kiteknologi takawas. Ontok 2011, kosuang o pogun diti sabaagi golongan pogun termakmur id pomogunan miimpohon do pendapatan per kapita. Pogun diti nga id taang kosiam nohopod miampai ginayo KDNK om id taang kotolu nohopod om onom tumanud kosulimbangan kemampuan berbelanja. Switzerlaand nogi nga id taang ko-20 tumanud ekspor, sundung do saiz nopo nga okoro. Nakaanu nogi rating takawas id Eropah montok Indeks Kobibasan Ekonomi 2010. Pondapatan per kapita pogun diti nga lobi akawas mantad kogumuan pogun Eropah Barat do suai om Jipun.

== Demografi ==
Poingiliu o Switzerland id pialatan do mogikaakawo koubasanan Eropah om minangarahung do pogun diti. Kiwaa tolu boros rasmi nasional ii nopo nga boros Jerman, Perancis om Itali, miampai iso boros rasmi momogun, boros Romansh. Pomorintaan persekutuan nopo nga wajib momoguno kotolu-tolu do boros rasmi dii om id parlimen nopo nga popotounda do mamaamadalin. Potiitikid mokiikinobos Switzerland nopo nga mositi minsingilo do iso mantad boros rasmi do suai gisom kogumuan mogiigiyon nga milo do dwiboros. Sabab do ogumu mogiigiyon om pakalaja asing (kiikiro 20%), boros Inggeris nogi nga asaru do gunoon. 
Kosoruan ugama id Switzerland nopo nga Katolik Rom soginumu do 45% mogiigiyon. Kiwaa nogi proteston kiikiro 35% om imigrasi ii minogowit do ugama Islam om Ortodoks Timur miampai do 2.5% om 1.5%.
