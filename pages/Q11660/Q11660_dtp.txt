Kabaalan tinanda (Boros Inggilis: artificial intelligence, AI) toi ponibaon do KATIN, nopo nga' iso hogot poinhakup do woyotoodo pamot mogilo (machine learning algorithms) om pomogunaan do woyotoodo dii. Poomitanan tagayo do kabaalan tinanda nopo nga' id somit boros tagayo (Large Language Model, LLM) toi konibaan dau, SOBOYO. Tongo somit diti pakayon id kikiit do ponguhup sasangai (virtual assistant) miagal Google Assistant om Siri. Okon nopo dii, mumu gumu nodi pomogunaan do kabaalan tinanda maya somit boros tagayo irad it ChatGPT om Gemini, ii tiitikid mamakai do soboyo GPT-3.5 toi GPT-4 don OpenAI, om somit Gemini don Google.


== Laguwagu ==
Kabaalan tinanda nopo nga' kipuun id goos do kikito dountokud (computer vision), om tantad dii mogikakawo do somit noworun miagal it somit pamotintong gumulapau (multilayer perceptron, MLP), pukot ravo sumulit (recurrent neural network), pukot ravo mikisus (convolutional neural network), om it woyowaal ponimban (Transformer architecture).

=== Pamakayan id gana' pongumbangan boros sandaudau ===
Pongumbangan boros sandaudau, POROSAN (natural language processing, NLP) nopo raan id kabaalan tinanda poingotul id ponutunan do boros tulun id ogumu voos ugu tuturan (speech), ginait (text) toi ko' sinuat, om nogi gumukawo (multimodal) ii momorolot lobi iso voos do boros tulun. Poomitanan tagayo nopo id kabaalan tinanda dompomorun (generative artificial intelligence), ugu mamakai boros tulun kivoos ginait do mamabang kumaa it somit do popoulai timot (image) toi ko' guguyu (video).
