Rusia, (Boros Rusia : Россия, tr. Rossiya,IPA: [rɐˈsʲijə], Jawi: ‏روسيا toi ko Pisompuruan Rusia nopo nga iso pogun rentas benua ii poingion id Eropah Kosilahon om Asia Koibutan. Miampai sompias 17,125,200 kilometer persegi (6,612,100 bt2), ii pogun do tagayo id pomoguon tumanud sompias, mangawalad lobi mantad iso perlapan koiyonon tana i ionon id Pomogunan. Dumimpot hopod om iso zon timpu, om miwolit miampai 16 pogun Watas. Watas Rusia poinhali mantad Rahat Baltik id kotonobon doid Rahat Pasifik id Kosilahon, om mantad Rahat Artik id koibutan doid Rahat Oitom om Caucasus id kabaatan. Miampai 146.7 riong mogigion poingiyon id suang 85 Subjek Pisompuruan pogun dii, Rusia nopo nga pogun bobos ginumu' mogigion id Eropah om kosiam id riniba. Kakadaian tagayo om ama' pogun Rusia nopo nga Moscow; koiyonon kakadaian apangkal i suai kaampai Saint Petersburg, Novosibirsk, Yekaterinburg, Nizhny Novgorod, Kazan om Chelyabinsk.
