Euro neɣ Yuru (Tamatart n tedrimt: €; ISO 4217: EUR), d tadrimt tunṣibt yellan di 20 n tmura seg 27 n tmura n Tdukli Tuṛufit.
Tamnaḍt-a n tmura tettwassen s yisem n Eurozone, tesɛa azal n 343 n yimelyunen n yimezdaɣen ɣef leḥsab n 2019.
Euro yella d tadrimt tameqqrant akk d tin yettwasuqlen deg ssuq n ubeddel n tedrimin tiberraniyin deffir n ddulaṛ n Marikan.
