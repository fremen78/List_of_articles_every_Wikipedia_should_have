Miksik nɣ Lmiksik (s tsbnyulit: México), ism amddud iga t Iwunak munnin n Miksikanus (s taṣbnyulit   Estados Unidos Mexicanos ) tga yat tagdudant tafidiralitn tamirikanit, tili ɣ unẓul n Iwankn Munnin n Marikan. Tg yat tmazirt lli mi iggut urky nns s takat n 120 mlyun n imzdaɣn. Tg tmdint n Miksiku tamaẓunt nns. 
Tga tamazirt n Miksik talli ɣ kullu ggutn middn lli isawaln tasblyunit. Tg Tis kraḍt immqqurn ɣ urky nns d tis snat ɣ tuggut n takat nns ɣ kullu marikan talatint. ɣ usggas n 2009 tga tdamsa nns tis snat ɣ tmurranin n tdamsa n umaḍal s yan PIB n 1482 n mlyar n Dular amirikani (USD)
