Tga Tinglizt (s tanglizt: English) yat tutlayt tajirmant ɣ tawja n tutlayin Tihindurufanin. Ar sis isawal wuṭṭun imqqurn n middn ɣ umaḍal, tg ula tutlayt tamzwarut ɣ kra n tmizar zun d Amirika, Britanya, Kanada, Ustralya, Nyu Zilanda,d Tfriqt n unẓul. Kigan n middn ɣ umaḍal ar sis sawaln, tg tutlayt tabṛṛaniyt tamzwarut g kigan n tmizar.
Tinglizt tga yat tutlayt tajirmanir d utrim, tffɣ d zɣ wawal lli s sawaln middn ɣ Britanya imqqurn, ɣ tmiḍi tiss 5, Inglizn, Isaksunn d Ijutn, n illan d iɣrfan d Ijirmanen, id ikkin zɣ uftis utrim n Ill n Ugafa, n illan d tilila gr n Uliman d Danmark. S manaya acku ntaf Tafrizyant, Talimant n wadday d Tinglizt mrawasnt gr asnt. Llan iḍrisn n tutlayin a zɣ usggwas n 700.
