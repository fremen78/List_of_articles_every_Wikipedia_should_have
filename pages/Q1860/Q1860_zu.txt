
== Ulimi lwesiNgisi ==
IsiNgisi (English) siwulimi olukhulunywa emhlabeni wonke ngabantu abangaphezu kwezigidi eziyizinkulungwane ezingu-1,5 njengolimi lwabo lokuqala, lwesibili noma lwesithathu. Njengoba luqale eNgilandi, ingeyomkhaya wezilimi zaseNtshonalanga-Germany futhi iye yashintsha phakathi namakhulu eminyaka, ithonywe isiLatini, isiFulentshi nezinye izilimi. Namuhla, ulimi olusetshenziswa kakhulu kwezamabhizinisi, kwezesayensi, kwezobuchwepheshe nakwezokungcebeleka. Ukusetshenziswa kwayo kabanzi nokusakazeka kwayo kwenza kube ithuluzi elibalulekile lokuxhumana nokushintshana ngamasiko.


== Isingeniso ==
IsiNgisi sinendima ebalulekile e-Afrika, sikhonza njengolimi oluhlanganisayo phakathi kokuhlukahluka okukhulu kwezilimi zezwekazi. Kwavela njengolimi lokuphathwa kwamakoloni, futhi nanamuhla, luhlala luyingxenye ebalulekile yezemfundo, ukubusa, kanye nokushintshaniswa kwamasiko.


== Umongo Ongokomlando ==
Ukusakazeka kwesiNgisi e-Afrika kusekelwe enkathini yamakoloni, ikakhulukazi ekoloni laseBrithani. IsiNgisi saba ulimi olusemthethweni ezizweni eziningi zase-Afrika, futhi leli fa lamakoloni laqhubeka nokwakha indima yalo ekubuseni nasebuhlotsheni bamazwe ngisho nangemva kwenkathi yenkululeko.


== Ukusatshalaliswa kwesiNgisi ==
IsiNgisi siwulimi olusemthethweni emazweni angaphezu kuka-20 ase-Afrika, kuhlanganise namazwe amakhulu anjengeNigeria, iNingizimu Afrika, iKenya neGhana. Ezindaweni ezinokuhlukahluka okukhulu kwezilimi, isiNgisi sivame ukusebenza njenge-lingua franca, sisize ukuvala izikhala zokuxhumana phakathi kwezikhulumi zezilimi zomdabu ezahlukahlukene.


== Indima Emfundweni ==
IsiNgisi sinendima ebalulekile ohlelweni lwezemfundo lwase-Afrika, kusukela emfundweni yamabanga aphansi kuya emfundweni ephakeme. Isebenza njengendlela eyinhloko yokuthola ulwazi nezinsiza zomhlaba wonke. Kodwa-ke, kunezinkinga, ikakhulukazi ezindaweni zasemakhaya lapho isiNgisi singakhulunywa khona kabanzi, ezithinta ikhono labafundi lokuqonda ngokugcwele okuqukethwe kwezemfundo futhi zikhawulele impumelelo yabo yezemfundo.


== Ukuphathwa Kwebhizinisi Namabhizinisi ==
Emazweni amaningi ase-Afrika, isiNgisi siwulimi olusemthethweni kuhulumeni, kwezomthetho nakwezamabhizinisi. Kwenza kube lula ukuba amazwe ahlangane, ahwebelane, futhi athengiselane, okwenza ukuba izizwe zase-Afrika zikwazi ukuhlanganyela ezimakethe zomhlaba. Kubasebenzi abaningi base-Afrika, ulwazi lwesiNgisi lubalulekile ekuthuthukiseni umsebenzi wabo nasekuzibandakanyeni emnothweni womhlaba.


== Umthelela Kwezamasiko ==
IsiNgisi sinethonya elikhulu ezincwadini, emculweni nakumabhayisikobho ase-Afrika. Abalobi abaningi abadumile base-Afrika nabenzi bamabhayisikobho basebenzisa isiNgisi ukuze bafinyelele izethameli eziningi emhlabeni wonke. Kodwa-ke, ukubusa kwesiNgisi ekukhiqizweni kwamasiko kuye kwaphakamisa ukukhathazeka ngokuncibilika kwezilimi zendabuko namasiko, njengoba izinkulumo eziningi zamasiko ase-Afrika zenziwa ngesiNgisi.


== Izinselele ==
Ukusetshenziswa kwesiNgisi e-Afrika kuletha izinselele ezifana nokungalingani kolimi, lapho ulwazi lwesiNgisi luvame ukuhlotshaniswa nesimo sezenhlalo nezomnotho. Amaqembu asengozini, ikakhulukazi ezindaweni zasemakhaya noma ezikude, angase abe nokufinyelela okulinganiselwe emfundweni ngesiNgisi, okuqhubekisela phambili imijikelezo yokungalingani. Ngaphezu kwalokho, kunokukhathazeka okwandayo ngokucwaswa kwezilimi zendabuko namasiko lapho kubhekene nokubusa kwesiNgisi.


== Amathuba ==
Naphezu kwezinselele, ukuba nolwazi lwesiNgisi kunikeza amathuba abalulekile kwezomnotho nawomsebenzi. Ivula iminyango yokuhwebelana kwamazwe ngamazwe, imfundo, nemisebenzi, inikeze abantu ithuba lokuxhumana nabantu emhlabeni wonke. Ukulinganisa ukusetshenziswa kabanzi kwesiNgisi nokulondolozwa nokukhuthazwa kwezilimi zendawo kubalulekile ukuqinisekisa ukuthi ifa elicebile lamasiko ase-Afrika alilahleki.
