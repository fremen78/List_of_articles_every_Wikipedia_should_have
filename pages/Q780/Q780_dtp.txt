Manuk (Ngaran kawo Linnaeus: Gallus gallus domesticus) nopo nga' iso dupot om saausontok (subspecies) do manuk talun taragang. Manuk nopo nga' iso mantad ogumu dupot nalaya pointongkop id riniba' soginumu lobi 19 bilion tinan di toun 2011. Manuk nopo piniara di tulun ponokotadon taakanon don tonsi' kaampai nogi do tontolu. Kosoruan, kotulud pia manuk om osomok nopo, om odomon om opiara o manuk.
Potudukon di ponoriukan papayat (genetic) kumaa ogumu tadon do sakag dau id Kabaaton Asia, Silaubaat, om Kosilahon, noruba nogi sanggamut (clade) dau id wanuo Mirika, Yurup, Tanga Silau om Apirika pointudan id Wanuokoro India. It kawo manuk odomon nooit mantad India kumaa Lidia id tonob do Asiakoro (Asia Minor), om kumaa Girik sumuang tatus kolimo Pogulu Tontok Puu (Before Common Era, BCE). It manuk notutunan id Kimit tantad tatus ko-15 Pogulu Tontok Puu, roitan do "tombolog tadau tadau maganak", om nooit mantad katanaan sorili, id Siria, Shinar, om Babylon ka di Susuyan di Thutmose III. Okuri kopio pisimbanan do poninan (genome) do manuk mikuo oduaki dau, it sawongraat kihulu (feathered dinosaur) i nounus tu' Kinounusan Kiritas-Ulailaid (Cretaceous-Paleogene extinction event).
