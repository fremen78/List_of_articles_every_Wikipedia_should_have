Timraradin (isem usnan: Reptilia) d iɣersiwen ikuẓdaren, Tinefrawin (amniote), Aqziz-is izzi-y-as wefrar.
Ar tura llan-t anagar 4 n tfesna: 

Tizutar (Crocodilia) deg-sent azal n 23 n telmas
Tiqerɣenbubin (Rhynchocephalia) deg-sent snat n telmas
Timeskebrin (Squamata) deg-sent azal n 7900 n telmas gar-asent izerman
Tifekrin (Testudines), Deg-sent azal n 300 n telmas


== Tilmas n temraradin ==
