Tga Taflsuft nɣ Taflsaft yat tɣarast n uswingm ɣ umaḍal, iɣzwr, d wamun. Tga tt asqsi n yan wanaw asilan n isqsitn f tumast (s tafransist d tanglizt : essence, nature) n uswingm n ufgan, d tin iɣzwr, d wassaɣn illan gratsn.
Kra n twal, ar sawaln middn f "tflsaft iẓlin s yan". Tga tt zɛma taɣarast nna s ittannay yan amaḍal. Aṛtikl ad ur igi f "tflsafin ẓlinin s middn", is iga f tswingmin nna f msawln inflsafn tasutin f tasutin.
Kra ɣ isqsitn nna d fkan inflsafn gan tn :

Ma'igan afulki (bien)?
Is illa umaḍal ad anɣ issutln?
Ma'igan taflsat?
Ma'igan tayri (amour)?


== Azur n yism ==
Taflsuft nɣ Taflsaft (neɣ :  Tayri n tussna) tga yat tguri d iwckan seg tutlayt Tagrikt n zik, seg (φιλοσοφία, Filusufya), taguri yad nzzḍar att nbḍu f snat tgzmin (Tigzmt tamzwarut :  φιλεῖν : Filyan : iri neɣ Tayri, tigzmt tiss snat : σοφία : Ṣufya : Tussna neɣ tinẓeẓt : الحكمة ). Ɣiladd nzzḍar anini mas Taflsuft dars anamk n :  Tayri n tussna neɣ Tayri n tinẓeẓt.
