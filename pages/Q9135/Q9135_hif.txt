Operating system (OS), ek system software hae jon ki computer hardware aur software ke sambhaale hae aur computer program ke sadhaaran sewa de hae. Operating system jon ke uu computer ke khaatir banaawa gais hae jiske dher jane baant (share) ke kaam me laae hae, kon task kab hoe ke chaahi, resource allocation aur jurra hardware ke sambhaale hae.
Computer hardware, jaise ki input aur output device aur kaise memory ke baata jaae hae, me OS program aur hardware ke biich me rahe hae. Operating system dher electronic chij me rahe hae, jaise ki mobile phone, video game console.
September 2023 me, personal computer ke market me Microsoft Windows ke 68% share rahaa, macoS ke 20% aur Linux aur ChromrOS ke 7%. Smartphone aur tablet ke market me Android ke 69% share hae aur iOS ke 30% hae. Linux, server aur supercomputer ke dominate kare hae.
