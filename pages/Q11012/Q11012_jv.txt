Robot ya iku piranti mékanik lan dhuweni kecerdasan gawéan kang bisa nglakokaké tugas kanthi nggunakaké pangawasan kontrol manungsa utawa nggunakaké program kang wis digawé kanthi dirancang dhisik. Robot bisa gerak dhéwé (otonom) uga bisa semi otonom lan bisa dikendalikake nganggo piranti liya sing dicekel utawa dikontrol manungsa. Robot lumrahé kanggo nglakokaké tugas kang abot kang ora bisa dilakoni manungsa. Saliyané abot pagawéyan kang dilakokaké robot uga mbebayani, dibolan-balèni, lan rusuh. Robot lumrahé digunanaké ing indhustri. Ing babagan indhustri robot lumrahé dianggo ing babagan prodhuksi. Robot ing indhustri uga duwé tugas ngresiki limbah kang duwé sipat racun, panjelajahan ing jero banyu utawa ing luar akasa, ing tambang, search and rescue, lan kanggo luru hasil tambang. Robot uga kena kanggo hiburan, pranti kanggo ngresiki omah kaya ta kanggo ngresiki awu, lan ngetoki suket.
 Saliyané iku robot uga manut nalika dikongkon. Ing jaman kang teknologi lan ngèlmu kang ngrembaka kanthi rikat bakal dirancang robot-robot kanggo nggampangaké tugasé manugsa. Robot uga ana kang bisa numpak montor. Nalika nggawé robot mbutuhaké ancangan kang bener saéngga robot bisa nglakoni apa kang wis diprogramaké. Kaya ta nalika robot dirancang kanggo lumayu aja nganti robot malah lumaku. Apamanéh nalika robot nggarap tugas kang wigati lan mbebayani, nalika salah program bakal nggawé rusak pagawéyan kang digarap. Nggawé robot mbutuhaké akèh piranti éléktronik lan komputer kanggo mrogram. Robot uga ora kena digawé sembarang wong, wong kang nggawé robot ya iku ahli mirunggan kang duwé kawruh kang mumpuni ing tèknologi lan éléktro.
Mesin awalé dianggo kanggo fungsi berulang (terus-menerus), kaya ta nimba banyu, ngetok wit lan liya-liyané. Kemajuan tèknologi mesin sing luwih kompleks mesin iku bisa dadi robot sing duwé kecerdasan gawéan, kanggo kabehe aktvitas pendukung manungsa, awit saka pekerjaan entheng nganti pagawéan abot, sing wigati nganti hiburan.


== Ikhtisar ==
Tembung Robot bisa kelbu robot mesing utawa sesuatu sing tampak, bisa dicekel lan ana wujudé, lan robot virtual utawa perangkat lunak sing dijenengi robot, nanging piranti robot virtual iku saiki dijenengi Bot (internet robot). Ora ana standar resmi sing memaktup apa waé sing dijenengi robot utawa luput. Wonten kesepakatan umum saka para ahli lan masarakat yèn robot iku cendrung ngelakokake adicara punika: bisa gerak lan gubah, bisa mengoprasikan apa waé sing kalebu fungsine, wonten kecerdasan gawéan, kecerdasan iku bisa niru manungsa, kéwan lan liya-liyané sing digarap déning manungsa utawa garapan saka kecerdasan olahanne dewe.
Robot iku tembang sing padinane subjektif.. Conto: Joseph Engelberger pionir saka Industri Robot tau ngomong, "Kula ora bisa mendefiniskan Robot, nanging kula bisa ngerti yèn piranti utawa benda iku robot." Miturut Ensiklopedia Britania, Robot iku "Kabèh mesin otomatis sing dioperasike sing genteni pirembagan manungsa nalika ora memper manungsa (segi fisik)". Merriam-Webster mendeskripsikan robot iku "Mesin sing memper manungsa lan bisa ngelakoake tindakan kompleks (bisa maca lan nulis) kaya manungsa", utawa "piranti sing otomatis ngelakoake tugas-tugas berulang lan rumit.


== Karakteristik ==
Nalika durung ana dhéfinisi tunggal kompleks saka tembung "robot" iku, nanging robot iku duwé beberapa utawa kabèh karakteristik.
iki ya iku mesin listrik sing dhuwei beberapa kaprigelan kanggo berinteraksi karo benda-benda fisik lan dikei kecerdasan gawéan kanggo ngelaksanakake aktivitas utawa fungsi tartamtu. mesin iki uga bisa nyerep objek fisik (nganggo sensor) ing lingkungan sakiwa-tengené, pengolahan data, lan timbal balik karo alam. iki sing bedakake robot karo komponèn mekanis prasaja utawa komponèn mekanis kompleks sing isane ngelakoake 1 utawa luwih fungsi.
