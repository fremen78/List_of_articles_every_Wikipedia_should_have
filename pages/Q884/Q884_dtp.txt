Korea Selatan (Korsel) toi ko' ngaran nokoimagon Republik Korea (RK, Hangul: 대한민국, Hanja: 大韓民國, Perumian Semakan: Daehan Minguk ) nopo nga iso pogun berdaulat id boogian kabaatan Korea id Asia Timur. Miwolit ii miampai Republik Rakyat China (Tanah Besar China) id kotonobon, Jepun id kosilahon om Republik Demokratik Rakyat Korea (Korea Utara) id koibutan. Ibu negara om kakadayan bobos agayo nopo nga Seoul. Korea Selatan nopo nga id kinoyonon di kiwaa iklim kebenuaan lembap om subtropika lembap miampai kouyuuyuho' di kinulu. Wilayahnya nopo dau nga popohompit kinoyonon kiginayo 100,363 kilometer persegi (38,750 bt2) om miampai mogigion soginumu 50 riong tulun, id nombo kakadayan Seoul nopo nga ionon do 10.4 riong tulun.
Asil noihum-ihum do arkeologi popokito do nokoumbal noion o Semenanjung Korea do tulun pogulu Zaman Paleolitik Awal. Nokoimpuun o Sajara Korea miampai ponuridangan Gojoseon ontok toun 2333 SM do raja Dan-gun i nosiliu lagenda. Gama pinisaan Tolu Koporintaan Korea id siriba pomorintaan di Silla ontok 668 M, pinorinta o Korea do Dinasti Goryeo (918–1392) om Dinasti Joseon (1392–1910). Nolimpupusan o pomorintaan Joseon soira diilhakkan o Korea do Empayar Jepun ontok toun 1910. Id kolimpupuson Sangod Pomogunan Koduo, binaagi o Korea kumaa  kuasa pendudukan Soviet id boogian koibutan om Amerika Syarikat id kabaatan. Sundung do tinuridong o Korea Selatan sabaagi pogun demokrasi ontok toun 1948, nga nasansagan ii monoguang kobolingkaangan politik om pomorintaan tentera i nokotilombus solinaid piipiro nohopod toun gisom noturidong o Republik Koonom ontok toun 1987.
