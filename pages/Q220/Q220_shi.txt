Tga Ruma tamdint tamaẓunt n Ṭṭalyan zɣ 1871. Tg ula tamdint lli akkʷ imqqurn ɣ tsga n Lazyu, s tsga tarakalt n Latyum. Tettubna iggi n wasif n Tibr, zdɣn gis kra n 2.8 id mlyun n umzdaɣ, yan usiḍn n OECD ar ittini is da ilkkem wuṭṭun n imzdaɣn n tsgiwin n tmdint attayn n 3.47 id mlyun. Lliɣ tsul tga talimbṛaṭurt n Roma tamaynut, ar aqqran i Roma s Tamdint lli ur ittmttatn acku tkka tt inn yad ɣ akud ann tga taqburt. Tkka 357 n usggʷas tga tamaẓunt n tmnkda tarumiyt. 
Tettyussan bahra Roma acku tga tgmmi n Tmẓẓalut Takatulikt, ula anssixf nns, walli mu ttinin Lbaba. Izdɣ ɣ tmdint n Vatikan lli igan yat tgzzumt ɣ iẓẓlmḍ-agmuḍan n tmdint n Ruma
