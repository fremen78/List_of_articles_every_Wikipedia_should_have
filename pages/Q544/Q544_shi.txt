Iga Angraw afukan nɣ Angraw anafuk (s tnglizt : Solar System, s tfransist : Système solaire) yan ungraw amtiwgan nna ɣ illa w-Akal. Gis yan itri, ad t-igan d tafukt, d kra n tɣawsiwin tignnawin ttuldaynin ɣ wattayn nns: tam n imtiwgn ittunẓan d wayyurn nnsn ittwassnen ganin ɣ uɣrud 185, smmus n imtiwgen mẓẓiyn d wayyurn nnsn ittwassnen d imlyaṛn n iqzzuyn imẓẓiyn (tugdt imqqurn n id astéroïde d imtiwgn yaḍn mẓẓiynin, les comètes, igḍrar igrmtiwganen, atg.). Angraw afukan yamu ɣ umahllaw igan s yism Abrid n walim. Yaggug i tuẓẓumt n umahllaw s wattayn n 8 kpc (∼26 100 a.l.).
