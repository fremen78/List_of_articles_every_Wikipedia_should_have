CPU nu mangrupa Céntral Procéssing Unit nyaéta mangrupa komponén teuas atawa perangkat hardware nu merosés data utama di jero hiji komputer. CPU ogé bisa disebut otakna komputer sabab CPU anu ngatur kabéh aktifitas jeung jalana kabéh program nu abus ogé kana aplikasi atawa  software di jerona.
Naon waé kabéh hal anu aya kakaitna jeung prosés dina komputer badé hal anu rumit nepikeun hal anu spélé pastina kaitanana jeung CPU. Contona lamun mencét huruf g, f, h atawa nu séjénna di kéyboard engkéna bakalan muncul dina layar CPU nu ngamungkinan éta téh kajadian.


== Komponen CPU ==
Suatu CPU teu boga hiji "prossésor" atawa suatu chip anu katanem di jerona anu fungsina ngolah kalkulasi komputer. Tapi saréng kamajuan téknologi anu beuki pesat ngamungkinkeun salah sahiji CPU boga leuwih ti procéssor atawa anu sok disebut "Procéssor core". Procéssor aya nu boga teknologi dual core, quad core, héxa core, octa core beuki loba jumlah core na beuki gancang ogé merosés data na, malahan lamun komputer anu leuwih canggih deui boga leuwih loba deui core na.


== Tugas CPU ==


=== Fetch ===
 Di aplikasi anu dijalankeun, loba paréntah anu kakirim kana CPU anu sumberna ti mémori program. Program éta téh di simpen dina alamat anu spésifik. Di jero CPU, aya anu ngaranna "Program Counter", éta téh tugasna keur ngitung posisi CPU di jero program.


=== Decode ===
Anu kudu anjeun nyaho nyaéta CPU ngan apal basa Assembly. Jadina, basa naon waé di gunakeun pikeun pemprograman anu anjeun paké di aplikasi bakalan dirobah kana bahsa Assembly. jeung bantuan "Assembler", basa Assembly dirobah kana kode biner. Jadina, CPU téh ngartos jeung bisa ngaéksekusi parentah anu dibere.


=== Execute ===
Dina parentah anu di bere ti prosés saméméha, CPU téh bisa ngalakukeun tilu hal sakaligus nyaéta: ngagunakeun ALU (Aritmethic Logic Unit), CPU bisa ngitung fungsi matematika nu kompléks, mindahkeun data ti memori ka lokasi anu lain, pindah ti hiji alamat dumasar kana anu di jieun ku CPU éta sorangan.


=== Writeback ===
Unggal waktu proséssor ngaeksekusi parentah, biasana bakal ngahsilkeun hiji kaluaran anu kudu diserat ulang. Saatos paréntah anggeus di eksekusi jeung ngahasilkeun hiji kaluaran, CPU bisa deui dipaké pikeun ngajankeun paréntah nu satuluyna.
