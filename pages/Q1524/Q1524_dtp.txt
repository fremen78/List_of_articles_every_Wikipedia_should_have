Athens (bahasa Yunani: Αθήνα, Athína) nopo nga ibu kota om bandar raya bobos pogun Yunani om kinoyonon sandad demokrasi. Kuta di pungaranan kopihondot do ngaran dewi Athena mimpohon mitologi, nopo nga sala iso bandar raya nokolohing id pomogunan, miampai sajara di nosuat okukurang do 3000 toun. Tadau baino, tama kuta dilo nopo nga iso metropolis kosmopolitan di atangkangau, miampai ginumu mogiigion bandar soginumu 3.1 riong tulun om ginumu mogiigion metropolitan soginumu 3.7 riong. Kinoyonon metropolitan Athens nopo nga longkod ekonomi, kousinan, perindustrian, koubasanan, om pulitik id Yunani. Osiisikap no om, nosiliu o bandar daya diti sabaagi iso longkod  pitaranan aagkal id Kesatuan Eropah. Ginayo kinoyon bandar di totoopot nopo nga 39 km², daamot dii aglomerasi bandar Athens nopo nga 412 km². 
Athens Kuno nopo nga iso negara kota di agaras kopio om longkod pambalajaran di poingaran-ngaran om tadon nogi di Akademi Plato om Lyceum Aristotle. Nokoinlaab o ponukuan o kinoyonon diti sabaagi kotimpuunon do  peradaban Barat, tu soboogian nopo do kinalantayon koubasanan om pulitik nga ontok ke-5 SM om ke-4 SM id Benua Eropah di noilaan ontok dii. Tinungkusan era klasiknya kakal do ajalas id bandaraya diti, om pinagambal maya piipiro monumen om karya inus kuno. I poimbantug nopo nga bongunan Parthenon id sawat do Akropolis ii no tu mercu tanda epik montok paradaban do barat . Athens nopo nga tuan walai montok Sukan Olimpik moden di kumoiso di toun 1896, om ontok toun 2004, Minanambut ii do Sukan Olimpik Musim Panas om nokoguli miampai kinalantayon bobos poimbantug.


== Ngaran ==

Id bahasa Yunani, ngaran Athens nopo nga Templat:Polytonic (Athenai), om mikait kumaa Templat:Polytonic (Athena) om Templat:Polytonic (Athene), oinsanan di nopo nga boros Yunani Attik om Ionia montok dewiAthena. Ngaran kuta diti nopo kanto nga id bontuk jamak, miagal Templat:Polytonic (Thebai) om Templat:Polytonic (Mykenai), tu ogumu o boogian. Ontok abad ke-19, Templat:Polytonic notorimo kawagu ngaran diti sabaagi ngaran bandar raya. Poingkuro po kalapas kotingkodon do kopomogunaan boros Greek nohimag o  Katharevousa ontok 1970-an, bontuk koizaai do Αθήνα (Athína) om nosiliu sabaagi ngaran poinhimag id bandar raya diti.

== Geografi dan jumlah penduduk ==
Pingromos o Athens id toinsanan arah dataran tanga Attica, asaru sukuon sabaagiLembangan Attica, di naantob do Nuluhon Aegaleo id kotonobon, Nuluhon Parnitha id koibutan, Nuluhon Penteli Koibutan Rahat, Nuluhon Hymettus id Kosilahon, om Teluk Saronic di Kotonobon daya. Nokoburu o Athens gisom Athens telah berkembang sehingga nokopuli do oinsanan rinantaion panakasabab do kosunian sumusuhut id maso dobontol apagon. Geomorfologi Athens panakasabab do nunu i roiton penyongsangan suhu di nontodon do soboogian kobolingkahangan ririnomos tayub di notimbaba do bandar raya wawagu diti.
