Kereta api (Jawi: ‏کريتا اڤي‎code: ms is deprecated ‎), atau tren ([trén], Jawi: ترين‎ pinjaman kata Inggeris train) terdiri daripada beberapa kenderaan berlandasan yang boleh bergerak bersama untuk membawa muatan atau penumpang dari satu tempat ke tempat yang lain melalui laluan tertentu. Landasan kereta api selalunya terdiri daripada rel konvensional. Pada masa kini, sesetengah kereta api menggunakan sistem monorel atau maglev. Daya tujahan kereta api datang dari pelbagai sumber. Biasanya lokomotif atau unit dorong diri (self-propelled unit) digunakan. Kereta api di Malaysia pada permulaannya digunakan bagi mengangkut bijih timah. Ia mula berkhidmat pada 1 Jun 1885 dari Taiping ke Port Weld.


== Jenis kereta api ==

Kereta api direka untuk memenuhi tujuan tertentu. Lihat Operasi pengangkutan kereta api.
Kereta api terdiri daripada gabungan satu lokomotif dan beberapa gerabak kereta, atau beberapa unit dorong diri (biasanya satu gerabak berkuasa yang dipanggil kereta rel). Kereta api juga boleh ditarik kuda, ditarik menggunakan tali, atau dibiarkan menuruni bukit (hasil tindakan graviti).
Kereta api istimewa diperlukan untuk landasan tertentu, misalnya monorel, kereta api laju, maglev, kereta api bawah tanah bertayar getah, kereta api kabel (funikulus) dan kereta api gunung. Kereta api penumpang terdiri daripada satu atau lebih lokomotif dan satu atau lebih koc ( coach ). Kereta api muatan pula terdiri daripada gerabak atau truk tetapi kereta api surat dan pos lebih menyerupai kereta api penumpang.kereta api yang dikemudi dengan dua lokomotif dinamakan "berkepala dua".
Kadang kala kereta api boleh bercampur penumpang dan muatan seperti di Mauritania. Namun, campuran sebegini agak pelik di kebanyakan negara. Kereta api istimewa digunakan untuk penyelenggaraan landasan.


== Tarikan ==

Pada mulanya kereta api ditarik kuda sehinggalah awal abad ke-19. Pada ketika itu engin wap digunakan. Pada 1920an, kereta api mula menggunakan diesel (sesetengahnya petrol), sehinggalah sekarang kereta api menggunakan tenaga elektrik. Pada 1970an kebanyakan negara telah mengganti kereta api wap untuk kegunaan harian. Hanya beberapa negara seperti China dan India masih menggunakan kereta api wap kerana sumber arang batunya yang murah dan banyak. Walau bagaimanapun kereta api jenis ini mula pupus. Kereta api bersejarah seperti ini masih digunakan di sesetengah negara sebagai salah satu daya tarikan pelancong. 
Lokomotif moden menggunakan engin diesel dan/atau motor elektrik. Biasanya, engin diesel akan memberi kuasa kepada motor elektrik untuk memutarkan tayar, atau sesetengah engin diesel menghantar kuasa ke tayarnya melalui kaedah hidrolik. Pemindahan mekanik, seperti di dalam kereta, digunakan pada sesetengah kereta api dan engin pirauan ( shunting engine ). Walau bagaimanapun, kos pengendalian kereta api diesel adalah tinggi. Oleh itu, jika kereta api itu mempunyai lalu lintas atau orang ramai yang mencukupi, kereta api itu mungkin akan menggunakan kuasa elektrik. Ini adalah kerana kereta api elektrik lebih murah untuk dikendali dan mempunyai prestasi yang lebih tinggi daripada kereta api diesel.


== Kereta api penumpang ==
Kereta api penumpang mempunyai kereta penumpang. Kereta api penumpang bergerak antara stesen, dan jarak antara stesen berbeza antara kurang daripada 1 km sehingga ribuan kilometer. 
Kereta api jarak jauh seperti Trans-Siberia mempunyai kereta restoran, kereta penginapan dan sebagainya. Kereta api jenis ini selalunya bergerak sederhana laju dan mengambil masa berhari hari untuk sampai ke destinasi. Manakala kereta api laju pula akan sampai ke destinasi sebelum waktu malam dan selalunya bersaing dengan kapal terbang.
Kereta api yang menghubungkan bandar bandar dinamakan kereta api antara bandar. Kereta api jenis ini tidak berhenti di stesen stesen kecil. Kereta api yang berhenti pada setiap stesen dinamakan kereta api setempat. 
Untuk jarak dekat di kawasan bandar, komuter mungkin digunakan. Sesetengah komuter direka supaya gerabak mempunyai lebih banyak ruang berdiri daripada duduk serta mempunyai kemudahan untuk basikal, kerusi roda, dan sebagainya. Sesetengah negara mempunyai kereta api penumpang dua tingkat yang digunakan dalam konurbasi. Di Eropah, kereta api laju dua tingkat semakin menjadi satu kebiasaan. 

Selalunya bandaraya bandaraya besar mempunyai sistem metro iaitu sistem kereta api bawah tanah yang turut dikenali sebagai subway di Amerika Syarikat, tube atau underground di United Kingdom atau metro di benua Eropah (Perancis, Belanda, Belgium dan sebagainya.) kereta api ini selalunya menggunakan kuasa elektrik pada rel ketiga dan landasannya berbeza dengan lalu lintas yang lain (iaitu tanpa lintasan sama aras). Selalunya metro menggunakan terowong dan kadangkala menggunakan landasan tinggi. Metro boleh memecut atau menyahpecut lebih pantas berbanding kereta api jarak jauh.
Kenderaan berlandasan yang menggunakan jalan raya dinamakan trem ( tram ), namun perbezaannya tidaklah ketara. Kadangkala istilah kereta api ringan diberikan kepada trem moden dan menyerupai metro kecuali kereta api jenis ini mempunyai lintasan sama aras ( level crossing ). Lintasan sebegini selalunya dilindungi pagar keselamatan.
Istilah rapid transit digunakan untuk pengangkutan awam seperti kereta api komuter, metro, dan kereta api ringan. Contohnya Putra LRT.
Kereta api maglev atau monorel merupakan teknologi baru dalam bidang kereta api. 
Kereta api penumpang selalunya mempunyai brek kecemasan. Penyalahgunaannya boleh dikenakan denda.


== Kereta api barangan angkut ==
Kebanyakan barangan dihantar menggunakan kereta api. Di sesetengah negara seperti AS, sistem kereta api digunakan sebagai pengangkut barang. Dalam sesetengah keadaan, mengangkut barangan menggunakan kereta api adalah sangat ekonomi, dan cekap tenaga (energy efficient) berbanding menggunakan jalan raya. Kereta api barangan angkut (KBA) adalah sangat ekonomi jika barang diangkut secara pukal pada jarak yang jauh. Penggunaan kereta api jenis ini adalah tidak berbaloi jika jaraknya dekat. 
Salah satu sebab mengapa KBA gagal menarik minat ramai orang ialah ketidakfleksibilitinya. Kebanyakan negara ketika ini cuba menggalakkan penggunaan KBA kerana sistem ini lebih mesra alam. 
Terdapat banyak jenis KBA. Antaranya ialah kereta api kontena, yang mana kontena diangkat menggunakan kren dan dibeban ke atas lori atau kapal laut.
