Karita api adalah tawas angkutan ril nang tadiri matan sambungan kandaraan nang ditarik sapanjang jalur karita api gasan ma'angkut kargu atawa panumpang. Gaya garak nang di'ulah ulih lukumutip nang tapisah atawa mutur indipidu dalam babarapa unit. Saupama parupulsi sajarah masin uap manduminasi, tawas-tawas mudirin nang paling umum adalah masin diasil qan listrik lukumutip, nang disadia'akan ulih kabal overhead atawa ril tambahan. Sumbar inirgi lain tamasuk kuda, tali atawa kawat, gravitasi, pnaumatik, batarai, wan turbin gas. Ril karita api rancaknya tadiri matan dua, talu atawa ampat ril, lawan sajumlah munuril wan guideways maglev dalam campuran. Kata 'train' asalnya matan Basa Prancis Kunu trahiner, lawan basa Latin trahere 'tarik, manarik'.


== Sajarah ==

Sajarah pakaritaapian sama nang kaya sajarah alat angkutan nang rancaknya dimula lawan panamuan ban. Mulanya dipinandu'i karita kuda nang wastu tadiri matan sabuting karita (sambungan), imbah ngitu di'ulah karita kuda nang manarik labih matan satu sambungan nang bajalan di jalur tatantu nang di'ulah matan wasi (ril) wan dingarani sapur. Ngini dipakai hususnya di dairah tambang wadah tadapat luri nang disambung wan ditarik lawan tanaga kuda.

Karita api ril kunpinsiunal adalah karita api nang rancak kita itihi. Mamakai ril nang tadiri matan 2 batang wasi nang di'andak di bantalan kayu jati nang karas.
