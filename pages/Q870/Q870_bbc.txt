Kareta api (Indonesia: Kereta api; Inggeris: Train) ima sada ulaula transportasi ni manisia jala boi dipangke museng mamboan angka barang-barang. Adong ragam ni kareta api, mamereng bahan na naeng padalanhon kareta api i. Sian haroroan diabad pa-19 hon, sude kareta api nungga mamangke lokomotip uap, na dibuat sian batu bara, hauma, manang miak lao padalanhon kareta api on. Dung i, hira-hira taon 1910, margonti ma bahan utama na, ima lokomotip diesel, dohot lokomotip listrik.

=== Segi teknologi jalur ===


==== Rel konpensional (dua batang besi) ====
Kareta api rel konpensional ima kareta api na bahat dipangke di Indonesia, songon kareta api na bahat di jalur Medan tu Rantau Prapat, songon kareta api Sri Bilah misal na. Kareta api on mamangke rel dua batang sian baja, na dipadosos  Menggunakan rel yang terdiri dari dua batang baja nadituntum anture dohot hau jati.


==== Rel margigi ====

Kareta api margigi on dibahen di luat na bahat rahis jala timbo, gabe dipangke ma rel namargigi, dipeakhon di tonga-tonga ni rel kareta api. Hagunaon rel namargigi on, asa pagogo kareta api tu rel nai, molo mardalan di luat na rahis. Holan adong di Pulo Sumatra dohot Pulo Jawa molo di Indonesia.


==== Monorel ====

Monorel, ima sistem transportasi kareta api na mamangke rel tunggal, manang holan sada rel. Monorel on adong dua ragamna, ima Monorel pelana, dohot Monorel gantung. Astunan na, monorel pelana ima kareta api na adong diginjang rel na, molo monorel gantung ima kareta api na adong di toruni rel, margantung di rel nai. Somalna, panumpang di kareta api on dang pola bahat, songon kareta api marrel konpensional. 


==== Maglev ====

Kareta maglev (Manglep) ima ragam ni kareta api na humusor mangareap manang mangaleap di ginjang ni rel. Somalna di hata Indonesia didok on ma Kereta Cepat, na boi mardalan hatop tu tujuan na. Na padalanhon kareta maglep on ima motor linear. Maglep dipangke gabe transportasi jarak na dao. Tingki dipadalan, dang tarbege suara mekanis ni kareta on. Daya tempuh ni maglep on boi sahat 500 km/jam. Adong museng pangambangon asa hatopna boi songon pesawat jet, ima 900 km/jam. 
Kareta ragam on dang adong dope di Indonesia. Angka di bangso na maju dope bahat Maglep on boi dijumpangi. Songon di Inggeris, kota Birmingham, adong ma maglep on dijumpangi. Angka bangso di Eropa, Amerika, Sina, Japang, nungga adong kareta on. Somalna karena on pe diarbishon tu bandara, gabe jalur utama transportasi haruar manang masuk tu bandara, khususna bandara internasional. Goar maglev dibuat sian hulimat marhata Inggeris ima magnetically levitated trains, astuanna kareta api nadiangkat sasara magnetis.


=== Segi propulsi (pangurdotna) ===

Adong opat ragama ni kareta api, mamereng pangurdotna. On ma ragam ni padalanhon kareta api:

Kareta api uap, ima kareta api na mamangke lokomotip uap.
Kareta api diesel, ima kareta api na mamangke lokomotip diesel.
Kareta api listrik, ima kareta api na mamangke lokomotip listrik.
Kareta api magnetik, ima kareta api na mamangke daya magnetik  lao padalanhon na, maglev.
