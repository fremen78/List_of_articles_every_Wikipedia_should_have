Belanda (Dutch: Nederland) toi ko' ngaran nokoimagon Kerajaan Belanda (Dutch: Koninkrijk der Nederlanden, "Kerajaan Tanah-Tanah Pamah") nopo nga iso pogun id koibutan kotonobon benua Eropah. Pogun nopo diti nga id piolitan do Jerman id kosilahon, Belgium id kabaatan om Laut Utara ponong id kotonobon piolitan maritim miampai United Kingdom.  Id aspek do kotoinaan, ii nopo nga pogun bobos agayo id suang Alam Belanda i porinta'on do porinta pogun dii om okon ko popohompit no kinoyonon id benua Eropah nga pohompit nogi piipiro pulou Caribbean ii no Bonaire, Sint Eustatius om Saba id Laut Caribbean.


== Pomungaranan ==
Ngaran Belanda nopo nga kitadon mantad boros Portugis  ii nopo nga Holanda, miampai do kopolombusan di alaab mantad do koiyonon poimbida id pantai barat pogun dii ii oroitan sabaagi do Holland.
